Create new user account

After you have clicked on  you will be taken to this form.
Legend
FieldDescription

username:

The system will suggest a username for the new account. However, you can choose any username that is not taken.

password:

The system will suggest a password for the new user account. However, you can enter any password you like. The user will later be able to change the password.
email:Enter the email address of the new user.
real name:Enter the name of the user.
set user active:Check to enable the user to log into the backend.

user is admin:

Check to give the user full administration rights.

verification:

Check to automatically send an email with the login details for the new account to the specified email address on clicking „send user data“.

Finalising account set-up

After you have filled in all fields please click this button



to create the new user account. The details will be save in the database.
