Content Part "Link & E-Mail"

direct link (optional): Here you enter the desired address (e.g. "http://www.google.com"). With an e-mail address, precede the address with "mailto:", for example "mailto:info@mydomain.com" - unfortunately only lowercase letters are possible here. If you need a link with capital letters, please use instead the content-part "link list".
target (optional): Select here, where the link is to be opened, for example "in a new
window". If you select nothing here, the link opens in the window in which the article
appears.
