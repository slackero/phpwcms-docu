Nachricht verfassen


Um eine Nachricht zu verfassen klicken sie auf "Neue Nachricht".
Sie sehen dann eine Übersicht die aus den Feldern,
Nachricht senden an, Verfügbare Empfänger, Betreff und Nachricht, besteht.

Das Feld Nachrichten senden an zeigt natürlich keine Empfänger an, durch Anklicken eines Names in Verfügbare Empfänger, wir dieser in das Feld Nachrichten senden an verschoben. Wenn Sie diesen Vorgang wiederholen können Sie eine Nachricht an mehrere Empfänger auswählen. 

Wenn Sie einen Empfänger aus der Liste Nachrichten senden an löschen wollen klicken Sie ihn erneut an. Dann wird dieser Empfänger aus der Liste gelöscht und erscheint wieder Verfügbare Empfänger. 

Im Feld Betreff tragen Sie den Betreff Ihrer Nachricht ein.

Nun Verfassen Sie im Feld Nachricht ihre Nachricht. 

Drücken Sie abschliessend auf um Ihre Nachricht abzuschicken.



Nach Drücken des Buttons erhalten Sie folgende Bestätigung, dass die Nachricht erfolgreich versendet wurde. 
Artikel erstellt: 26. November 2004 Letzte Änderung: 25. Januar 2005
