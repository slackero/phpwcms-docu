Content Part "E-Card"

selector: Select between radio buttons and JavaScript functionality. With JavaScript functionality you can use the following fields:

- onMouseOver
- onClick
- onMouseOut

image: Choose here the images that you want to offer as an e-card.

alignment: The following alignments are available:
left - center - right

max. width - max. height:  Insert here the size of the e-card images in pixels.

column:  You can choose up to 10 columns for the alignment of the images.

image space: If you want to show the images among each other, you have to insert here the space value in pixels.

 

Click enlarge: Click this option when you want to give the user the possibility to enlarge an image.

title/e-card:  Insert here the title of the e-card.

form tmpl: Here you can define the form that prompts the user for the sender's and recipient's data as well as for the message to send.
The following placeholders can be used in the form:

###ECARD_SUBJECT###: The title of the message to send
###SENDER_NAME###: The sender's name
###SENDE_EMAIL###: The sender's email address
###RECIPIENT_NAME###: The recipient's name
###RECIPIENT_EMAIL###: The recipient's email address
###SENDER_MESSAGE###: The sender's message for the recipient
<!--FORM_ERROR_START-->: Start tag for the error handling
<!--FORM_ERROR_END-->: End tag for the error handling

sent tmpl:  This field can you use for the creation of a success message that appears when the e-card has been successfully sent.
The following placeholders can be used in the sent form:

 

 

###ECARD_SUBJECT###: The title of the sent message
###RECIPIENT_NAME###: The recipient's name
###RECIPINT_EMAIL###: The recipient's email address
###SENDER_MESSAGE###: The sender's message for the recipient
###ECARD_TITLE###:  The e-card title
###ECARD_IMAGE###: The e-card image

mail tmpl: Here you can define the email that the recipient of the e-card will receive.
The following placeholders can be used in the email form:

###ECARD_SUBJECT###: The title of the received e-card
###SENDER_NAME###: The sender's name
###SENDE_EMAIL###: The sender's email address
###RECIPIENT_NAME###: The recipient's name
###RECIPIENT_EMAIL###: The recipient's email address
###SENDER_MESSAGE###: The sender's message for the recipient
###ECARD_IMAGE###: The e-card image
###ECARD_TITLE###:  The e-card title
