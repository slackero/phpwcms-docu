Nachricht verfassen

Bild 1 - Neue Nachricht erstellen

Sie sehen hier (Bild 1) eine Übersicht die aus den Feldern, Nachricht senden an, Verfügbare Empfänger, Betreff und Nachricht, besteht.

Das Feld Nachrichten senden an zeigt natürlich keine Empfänger an, durch Anklicken eines Names in Verfügbare Empfänger, wir dieser in das Feld Nachrichten senden an verschoben. Wenn Sie diesen Vorgang wiederholen können Sie eine Nachricht an mehrere Empfänger auswählen. 

Wenn Sie einen Empfänger aus der Liste Nachrichten senden an löschen wollen können Sie den Empfänger mit einem Doppelklick entfernen oder Sie klicken Ihn einfach an und benutzen dann das Symbol . Dann wird dieser Empfänger aus der Liste "Nachricht senden an" gelöscht und erscheint wieder Verfügbare Empfänger. 

Im Feld Betreff tragen Sie den Betreff Ihrer Nachricht ein, im Feld Nachricht den gewünschten Text.  Drücken Sie abschliessend auf um Ihre Nachricht abzuschicken.

Bild 2 - Nachricht versendet

Nach Drücken des Buttons erhalten Sie folgende Bestätigung (Bild 2), dass die Nachricht erfolgreich versendet wurde. Sie haben dann die Möglichkeit auf "Neue Nachricht erstellen" zu klicken, um eine neue Nachricht zu erstellen. Ansonsten werden Sie Automatisch weitergeleitet zurück auf die Nachrichtenzentrale.
