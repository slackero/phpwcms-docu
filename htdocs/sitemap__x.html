Content Part "Sitemap"

Mit diesem Content Part ist es möglich, eine sogenannte Sitemap Ihren Besuchern zur Verfügung zu stellen.
Davor (optional): Hier können Sie den Text eingeben, der vor der Sitemap erscheinen soll.
Danach(optional): Hier können Sie den Text eingeben, der nach der Sitemap erscheinen soll.

Ebene Icon (optional):

 Hier können Sie Ihr eigenes Bild für die Ebenen auswählen, z.B. img/article/fill_in_here.gif  für
Artikel Icon (optional): Hier können Sie Ihr eigenes Bild für die Artikel auswählen, z.B. img/usricon/usr_admin.gif für
Anzeigen:
Hier können Sie eine der folgenden Optionen auswählen:
1. Nur Seitenebenen: Die erzeugte Sitemap zeigt nur die Seitenebenen an.
2. Seitenebenen und Artikel: Die erzeugte Sitemap zeigt sowohl Seitenebenen als auch Artikel an. 
Ebene Class (optional): Erzeugen Sie eine Klasse für die Seitenebenen in der frontend.css und geben Sie nur den Klassennamen in dieses Feld ein.
Artikel Class (optional): Erzeugen Sie eine Klasse für die Artikelansicht in der frontend.css und geben Sie nur den Klassennamen in dieses Feld ein.
Zähler: Hier können Sie eine der folgenden Optionen auswählen:
1. nicht im Class-Name mitführen: (Keine Daten verfügbar)
2. im Class-Name mitführen: (Keine Daten verfügbar)
