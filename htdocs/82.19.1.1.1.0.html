Die Artikelzentrale

Im Bereich „Artikel“ sehen Sie zunächst die Artikelzentrale – eine Übersicht aller auf der Website veröffentlichten Artikel und Strukturebenen. Diese werden innerhalb der Struktur der Website ähnlich der Darstellung von Ordnern im Windows Explorer dargestellt.

 

Die Artikelzentrale im Detail

Von der Artikelzentrale aus können Sie sich nicht nur einen Überblick über alles verschaffen sondern auch Artikel in Strukturebenen erstellen, kopieren und löschen. Ausserdem können Sie auch direkt auf Contentparts zugreifen.

 

Um direkt auf die Contentparts zuzugreifen klicken Sie auf das Sie vor dem finden. Sie sehen dann eine Übersicht alle benutzen Contentparts. 

Unterhalb des Artikels sieht man nun eine neue Ebene. Jeder Zeile beginnend mit ist ein Contentpart des Artikels zugeordnet. Danach wird die ID des Contentparts in [ ] angezeigt, ausserdem welcher Contentpart typ es ist und zum schluß wird mit angezeigt in welchem Bereich des Template/Layout  der Inhalt angezeigt wird.
