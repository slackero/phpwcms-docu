Graphical Text MOD

Übersicht

Das Graphical Text MOD ist eine Erweiterung, welche es Ihnen ermöglicht, zur Laufzeit mittels Replacement Tags Text ein vorher fest definiertes Format zuzuweisen. Das Besondere an dieser Methode ist, dass Sie jeden Truetype Font benutzen können um damit Überschriften oder Navigationen zu gestalten. Der gewünschte Text wird mit einer von phpwcms generierten Grafik ersetzt. Um die Last für den Webserver zu vermindern, verfügt das System über einen Cache, der einmal generierte Grafiken speichert und dann bei erneutem Aufruf neu lädt.

Auf den folgenden Seiten bekommen Sie eine kurze Übersicht über die Funktionsweise dieses MODs, welches bisher als einzigster fest in phpwcms integriert wurde.

Wenn sie auf Module klicken und danach Graphical Text MOD dann können Sie damit beginnen, Schriften, Farben und Stile zu definieren.

Bitte beachten Sie, dass folgende Vorraussetzungen gegeben sein müssen:

  • Trutype Schriftarten (*.ttf) müssen per FTP in den Ordner include/inc_module/mod_graphical_text/inc_fonts/ geladen werden.
  • das Unterverzeichnis gt muss im Verzeichnis /content vorhanden sein und mittels CHMOD auf 777 gesetzt werden.

 

Schriftarten definieren

Wenn Sie Schriftarten per FTP in das oben angegebene Verzeichnis kopiert haben und auf Schriftarten klicken sehen Sie eventuell schon definierte Schriften und die von Ihnen neu heraufgeladenen Schriften mit dem Zusatz noch nicht hinzugefügt.

Klicken Sie auf  um die Schrift dem System hinzuzufügen. 
Name: Klartextbezeichnung für die Schrift.
Kurzname: Eine Kurzbezeichnung der Schrift
Aktualsieren: Diese Daten übernehmen und die Schrift dem System hinzufügen
Abbrechen: Alle Änderungen verwerfen und Schrift nicht hinzufügen.

Nach dem Klicken auf Aktualisieren kehren Sie zurück zur Übersicht und es wird Ihnen zu der neu hinzugefügten Schrift eine Schriftprobe angezeigt.
