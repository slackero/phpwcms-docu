Content Part "Liste"

Mit dem Content Part "Liste" können Sie Aufzählungslisten definieren.

Die unterschiedlichen Ebenen einer Aufzählungsliste können Sie durch das ~-Zeichen (Tilde) darstellen, d.h.

~ : 1. Ebene
~~ : 2. Ebene
~~~ : 3. Ebene
~~~~ : 4. Ebene
usw.

Beispiel:
  • 1. Ebene 1
    • 1.1. Ebene 2
      • 1.1.1 Ebene 3
        • 1.1.1.1 Ebene 4
    • 1.2. Ebene 2
      • 1.2.1 Ebene 3
        • 1.2.1.1 Ebene 4
  • 2. Ebene 1
    • 2.1. Ebene 2
      • 2.1.1 Ebene 3
        • 2.1.1.1 Ebene 4
    • 2.2. Ebene 2
      • 2.2.1 Ebene 3
        • 2.2.1.1 Ebene 4
