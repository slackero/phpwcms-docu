Verwaltung des eigenen Profils



Im Profil können die eigenen persönlichen Daten jedes Backend-Users angepasst werden.

Zur Sicherheit sollte das Passwort in regelmässigen Abständen, mindestens jedoch alle 2 Monate gewechselt werden. Bitte verwenden Sie keine Wörter, die in jeglichen Wörterbüchern jeglicher Sprachen und Dialekte auffindbar sind. Verwenden sie stattdessen zusammenhangslose Kombinationen aus Zahlen und Buchstaben.

Das Feld Email sollte auf jeden Fall eine gültige Email-Adresse enthalten, da an diese Adresse sämtliche Änderungen im System mitgeteilt werden. Diese Adresse ist ausschließlich für Backend-User sichtbar und somit keinem Spambot zugänglich.


Die wichtigste Einstellung im Menupunkt Profil ist die Sprache. Wählen Sie hier die Sprache, die Sie am besten beherrschen und speichern Sie ihre Änderungen ab. Von nun an wird das Backend bei jedem Login in der von Ihnen gewählten Sprache erscheinen, völlig egal, welche Sprache das Dropdown-Feld auf der Login-Seite enthält.



Auf der linken Seite haben Sie die Möglichkeit aus dem Untermenu den Punkt "Persönliche Daten" auszuwählen. Dieser Menupunkt ist selbsterklärend, der wichtigste Hinweis bezüglich der Datensicherheit befindet sich oben auf der Seite. Mit "andere Benutzer" sind hier lediglich die Backend-Benutzer gemeint.
