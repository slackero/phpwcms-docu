Spracheinstellungen für Replacement Tags

Einstellungen für die DATE und TIME tags

Sie finden in der (config/phpwcms/) conf.template_default.inc.php ab Zeile 158 
einmal die Möglichkeit die Sprache zu ändern. Dafür tragen Sie anstatt des Kürzels DE das Kürzel der gewünschten Landessprache ein.

Des Weiteren haben Sie darunter noch die Möglichkeit das Datumsformat  zu verändern.

// date and time formatting
$template_default["date"]["language"]  = "DE";          // DE=German, IT=Italian, FR=French
$template_default["date"]["long"]   = "l, j. F Y";          // (Monday, 1. October 2003)
$template_default["date"]["medium"]   = "D, j. M y";  // (Mon, 1. Oct 03)
$template_default["date"]["short"]   = "Y/m/d";          // (2003/12/25)
$template_default["date"]["article"]  = "Y/m/d";         // (2003/12/25)
$template_default["time"]["long"]   = "H:i:s";             // 15:25:45
$template_default["time"]["short"]   = "H:i";              // 15:25

Einstellungen für den {NEW:X} TAG

Auch für den {NEW:X} Tag haben Sie die Möglichkeit das Datumsformat und die Sprache zu verändern.

Dazu gehen Sie bitte wieder in die (config/phpwcms/} conf.template_default.inc.php Datei zur Zeile 189. Dort finden Sie folgende Zeile:

$template_default["news"]["date_language"] = "EN";

Ändern Sie das Kürzel in die Sprache in der Sie es ausgeben wollen.
