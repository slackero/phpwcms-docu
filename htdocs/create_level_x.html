Create new structure
To create a new category click in the site structure overview which will take you to this form.

Legend
Field Describtion
category title: Title of the category.
frontend menu status: hide – Category will not be displayed in the site navigation menu.
visible for users logged on only – This option has no function at the moment.
alias this category: Enter an alias for the system to call on, or if you are using the rewrite function to create a document name for this category.

Exmaples:
www.domain.com/index.php?alias 

or 

www.domain.com/alias.html (with rewrite)
redirect to: Gives you the possibility to redirect to a different category or an external site. To redirect to an internal category we recommend you use the {SITE} tag.
 
To redirect to an external source simply enter the URL of the site followed by a space and ‘_blank’. 

Note: for internal redirection always use this format: index.php?alias
category infotext: Here you can enter a description for the category.
template: Choose a template on which the new category should be based on.
top article count: Number of articles that will be displayed with summery in the list view.
ordering of assigned articles: Determines the order in which the articles are displayed in the list view.

Available options:

manual (arrow up/down)
creation date
change date
start date

In addition you can choose:

ascending
descending
status: In order to make the category visible for your visitors tick both visible and public.
