Datenbank sichern
Um die Datenbank zu sichern können Sie das von den meisten Providern zur Verfügung gestellte phpMyAdmin benutzen. Wie der Pfad zu diesem Programm ist erfahren Sie von Ihrem Provider.

Die folgenden Screenshots können sich je nach Version von phpMyAdmin unterscheiden. Die Vorgehensweise bleibt jedoch identisch.


Wenn Sie phpMyAdmin aufgerufen haben sehen Sie als erstes diesen Bildschirm. Klicken Sie auf den Datenbanknamen um zum nächsten Schritt zu gelangen. Der Datenbank name ist auf dem Foto rot umrandet.











Um zu den Datenbanksicherungseinstellungen zu gelangen, drücken Sie jetzt bitte auf Exportieren.













Um eine komplette Datenbanksicherung zu erstellen, befolgen Sie bitte folgende Schritte.

Im rechten Bildschim sehen Sie das Feld Exportieren. Wählen Sie dort  Alles markieren aus. Oder Sie wählen nur die phpWCMS Tabellen aus. Die können sie am besten markieren indem Sie die erste phpWCMS Tabelle anklicken und dann die Taste Shift gedrückt halten und mit der Pfeil nach unten Taste solange nach unten drücken bis alle phpWCMS Tabellen markiert sind.



Dann wählen Sie rechts unter SQL Optionen

Mit Drop Table aus. Das bewirkt dass die vorhandenen phpWCMS Tabellen überschrieben werden.

Des Weiteren sollte bei Daten ein Haken gesetzt sein.

Dann wählen Sie unter Kompression noch gzip aus.

Und drücken dann auf OK nun wird ein Zipfile erstellt welches Ihre Datenbanksicherung enthält.
