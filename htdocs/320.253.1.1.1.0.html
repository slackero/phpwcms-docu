Benutzerverwaltung

Benutzer

anlegen und verwalten

In der Benutzerverwaltung können Sie neue Benutzer anlegen. Sie können hierbei entscheiden ob der neue Benutzer einfache Benutzerrechte oder aber Administratorrechte erhält.

Wenn Sie das erste Mal die Benutzerverwaltung anklicken werden Sie nicht mehr als den Webmaster als Benutzer sehen.

Nachfolgend erhalten Sie einen Überblick über alle Icons in der Benutzerverwaltung.
Legende
IconBeschreibung

An diesem Icon können Sie sehen, dass der Benutzer Administratorrechte hat.

An diesem Icon können Sie sehen, dass der Benutzer den Benutzerstatus hat und somit keinen Zugang zum ADMIN Bereich.

Dieses Icon zeigt den Status des Benutzers an. 

bedeutet: Der Benutzer kann sich anmelden.
bedeutet: Der Benutzer kann sich nicht anmelden.

Durch Anklicken von oder  können Sie diesen Status ändern.

Wenn Sie auf dieses Icon drücken gelangen Sie zum gleichen Eingabebildschirm wie beim Benutzer anlegen und können dort dann den Namen, das Passwort  oder die E-mail Adresse ändern.

Durch Drücken auf dieses Icon löschen Sie den Benutzer aus der Benutzerliste.

Durch Klicken auf dieses Feld gelangen Sie zur Eingabemaske um einen neuen Benutzer zu erstellen.
