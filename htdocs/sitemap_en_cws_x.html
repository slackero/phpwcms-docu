Content Part "Sitemap"

With this content part you can offer your visitors a sitemap.

before (optional): Here you enter the text that appears before the sitemap
after (optional): Here you enter the text that appears after the sitemap

level icon (optional):

 Here can you choose your own image for the levels e.g.  img/article/fill_in_here.gif  for
article icon (optional): Here you can choose your own image for the articles e.g. 
img/usricon/usr_admin.gif for
display:
Here you select from one of the following options:
1. structure levels only: the generated sitemap will only display the structure levels
2. structure levels + articles: the generated sitemap will display both structure levels and articles 
level class (optional): Create a class in the frontend.css for the levels and type only the class name in this field.
article class (optional): Create a class in the frontend.css for the article view and type only the class name into this field.
counter: Here you select from one of the following options:
1. don’t add to class name: (no data available)
2. add to class name: (no data available)
