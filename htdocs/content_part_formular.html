Content Part "Formular"

früher "E-Mail Kontaktformular"

Hinweis:
Dieser Content Part "E-Mail Kontaktformular" kam mit der Version 1.2.
Mit der Version 1.2.9. wurde er umbenannt in "Formular"

Content Part "Formular"

Hier finden Sie die Beschreibung wie Sie
ein E-Mail Kontaktformular einrichten können.

In älteren Versionen von phpwcms wurde dieser content Part
als E-Mail Kontaktformular bezeichnet.


Nachfolgend  eine Übersicht der zur Verfügung stehenden
Eingabe-Felder.
Feldbeschreibungen
Betreff: Geben Sie hier den Betreff ein, der in der E-Mail an den Empfänger (Ziel) und eventuell den Absender (Kopie an) übermittelt werden soll.
Ziel:
 Hier geben Sie den Empfänger der Email an.
Kopie an: Soll der Absender auch eine Kopie erhalten, wählen Sie bitte hier das Formularfeld aus, in dem die Absender-Email-Adresse steht.
bei Erfolg: Soll der Absender bei einer erfolgreich übermittelten Email eine Meldung erhalten, geben Sie bitte hier den Alias der Seite ein, die die Meldung enthält.
bei Fehler: Soll der Absender bei einer fehlerhaft übermittelten Email eine Meldung erhalten, geben Sie bitte hier den Alias der Seite ein, die die Meldung enthält.
Formular: In den folgenden Feldern können Sie das Aussehen des Email-Kontaktformulars definieren.
Anordnung: Hier können Sie die Anordnung von Label und Feld definieren. Folgende Möglichkeiten stehen zur Auswahl: 1. Label links vom Eingabefeld
2. Label oberhalb des Eingabefeldes 
Formular class: Soll das Formular mit einer speziellen CSS-Klasse angezeigt werden, fügen Sie den Namen der Klasse bitte hier ein.
Label wrap: Soll das Label speziell gewrappt werden, geben Sie bitte die Daten hier ein. Das Zeichen "|" dient als Labeltextplatzhalter.
Bsp.: |&nbsp;&nbsp;
(Fügt einen Abstand von zwei Zeerzeichen zwischen dem längsten Label und dem Eingabefeld ein, bei horizontaler Ausrichtung).
Pflichtzeichen:  Pflichtfelder können besonders gekennzeichnet werden. Die Standardkennzeichnung ist *.
Fehler class: Soll die Fehlermeldung mit einer speziellen CSS-Klasse angezeigt werden, fügen Sie den Namen der Klasse bitte hier ein.
Typ:

Es können unterschiedliche Feldtypen in das E-Mail Kontaktformular eingepflegt werden. Die Befüllung der Feldtypen erfolgt über das Eingabefeld "Wert".

Folgende Feldtypen sind möglich: 

Text (einzeilig)
Ein einzeiliges Textfeld wird eingeblendet.  

Text ( mehrzeilig)
Eine Textarea wird eingeblendet.  

Text (spezial)
Ein einzeiliges Textfeld mit z.B. einer Formel oder einer Variablendefinition wird eingeblendet.  

Versteckt
Verwendung eines versteckten Feldes.  

Passwort
Anzeige eines Passwortfeldes, in dem die Passworteingabe verschlüsselt anzeigt wird.  

Email
Anzeige eines Email-Eingabefeldes.  

Ausklappmenü
Anzeige eines DropDown-Menüs.  

Liste Anzeige einer Auswahlliste.   

Kontrollkästchen
Anzeige von einer oder mehreren Checkboxen. Wenn Sie einen Wert vorbelegen möchten, schreiben Sie bitte direkt hinter den betreffenden Wert ein Leerzeichen sowie das Wort "checked", d.h. " checked".  

Optionsschalter
Anzeige eines oder mehrerer Radiobuttons. Wenn Sie einen Wert vorbelegen möchten, schreiben Sie bitte direkt hinter den betreffenden Wert ein Leerzeichen sowie das Wort "checked", d.h. " checked" .  

Datei
Hier besteht die Möglichkeit, den Benutzer eine Datei aus seinem Dateisystem auswählen zu lassen. Es erscheint ein Feld für den Dateinamen sowie ein Durchsuchen-Schaltknopf. 

Senden-Taste
Anzeige eines Senden-Schaltknopf. Die Beschriftung des Schaltknopfes kann beliebig eingegeben werden.  

Zurücksetzen
Anzeige eines Reset-Schaltknopf. Die Beschriftung des Schaltknopfes kann beliebig eingegeben werden.  

Trenner
Anzeige von Bereichstrennern. Eingabe beliebig. 

Zwischentext
Eingabe eines Zwischentextes.
Name: Tragen Sie hier einen eindeutigen Feldnamen ein.
Label: Hier müssen Sie den Labelnamen eingeben, der dem Benutzer links neben dem Eingabefeld oder oberhalb des Eingabefeldes angezeigt wird (s.a. Feld "Anordnung").
S/C: In diesem Feld können Sie für - einzeilige Textfelder die Größe und für - mehrzeilige Textfelder die Zeilenanzahl eingeben.
M/R: In diesem Feld können Sie für - einzeilige Textfelder die maximale Länge und für - mehrzeilige Textfelder und Listen die Spaltenanzahl eingeben. Bei Listen bewirkt eine Eingabe des Wertes "1" in dieses Feld, dass Sie mehrere Listeneinträge auf einmal auswählen können (Mehrfachselektion). Die Mehrfachselektion können Sie mit der STRG-Taste durchführen.
! (Ausrufezeichen): Dieses Feld kennzeichnet den linksstehenden Feldtypen als zwingend erforderlich.
Papierkorb: Mit dem Papierkorb können Sie den linksstehenden Feldtypen löschen.
Wert:

Hier können Sie Werte für die Feldtypen eingegeben:

Text (einzeilig)
"Das ist ein einzeiliger Text" 

Text ( mehrzeilig)
"Das ist ein mehrzeiliger Text" "Das ist ein mehrzeiliger Text" 

Text (spezial)
type=INT - nur Ganzzahlwerte zulässig
type=DEC oder FLOAT - nur Dezimal bzw. Float zulässig
type=IDENT - Vergleichsprüfung, nur wenn Eingabe identisch mit dem default Wert
type=STRING oder MIX
type=DATE - prüft ob gültiges Datum
type= A-Z (nur Großbuchstaben)
type=a-z (nur Kleinbuchstaben)
type=a-Z (nur Buchstaben)
type=0-9 (nur Ziffern)
type=PHONE (Telefonnummer)
type=WORD (Wortbuchstaben)
type=LETTER+SPACE (a-Z, space, hyphens)
type=REGEX (patterns - new value: pattern

CSS-Stil: Auch für diese Fehlermeldung können Sie eine individuelles CSS-Klasse verwenden.
Vorlage: Hier muß ein Template eingegeben werden, sonst erhalten Sie eine leere Email zugesendet. Dieses Template kann als Text-Template oder als HTML-Template erstellt werden. In den Templates können Sie alle Namensfelder des Formulars verwenden.
