Nachricht lesen
und beantworten


Wenn Sie eine Interne Nachricht erhalten haben sehen Sie, dass sofort nach dem Einloggen in das System oben rechts im roten Balken sehen Sie dieses


Symbol. Jetzt haben Sie 2 Möglichkeiten die Nachricht aufzurufen. Entweder Sie klicken das Symbol an oder Sie klicken in der gleichen Zeile auf "Nachricht"

Nachrichten Überblick



Nachdem Sie nun auf eine der beiden Möglichkeiten geklickt haben, gelangen Sie zur Nachrichten Übersicht. Dort können sehen Sie nun wer Ihnen eine Nachricht und mit welchem Betreff gesendet hat. Am Ende der Zeile sehen Sie 2 Icons.

Das erste Icon leitet Sie zur Antwortseite weiter.
Das zweite Icon löscht die Nachricht. Gelöschte Nachrichten werden aber nicht sofort unwiderbringlich gelöscht, sondern werden in den Ordner "Gelöscht" verschoben.

Wenn Sie jetzt entweder auf den Absender der Nachricht oder den Betreff klicken können Sie die Nachricht lesen.


Nachricht lesen



Hier können Sie nun die Ihre Nachricht lesen. Unter der Nachricht sehen Sie 4 Schaltflächen.

Das sind die Bedeutungen der einzelnen Schaltflächen:
 
Schließen Sie die Nachricht
Erstellen einer neuen Nachricht ohne auf die Nachricht die Sie gerade lesen Bezug zu nehmen.
Gelangen Sie zur Antwortseite
    Löscht die Nachricht.

Nachricht beantworten



Hier können Sie nun die Nachricht beantworten. Es funktioniert genauso wie wenn Sie eine Nachricht verfassen, mit der einzigen Änderung, dass Sie hier keinen Empfänger auswählen müssen.
Artikel erstellt: 27. November 2004 Letzte Änderung: 25. Januar 2005
