Content Part "Angebot"

Hier finden Sie die Beschreibung wie man ein Angebot einrichtet. Sie sehen hier eine Übersicht aller Felder. Im Anschluss daran können Sie die Beispieltexte gern für Ihre Seite verwenden.

 

Feldbeschreibungen

Bild: Hier können Sie ein Bild zu Ihrem Angebot auswählen.
Startdatum:
&
Enddatum: 		Bestimmen Sie hier das Start- und Enddatum Ihres Angebotes. Das Format für beide Felder sieht folgendermaßen aus: (Jahr-Monat-Tag Stunde:Minute:Sekunden) 2004-12-24 00:00:00

Startgebot:
&
erhöhen um:
Bestimmen Sie hier wie hoch das Startgebot ist und um welchen Schritt es sich nach der Gebotsabgabe erhöhen soll.
Beispiel: 10€ Startgebot 1€ erhöhen = Mindestgebot 11 €
Davor: Hier können Sie noch einen einleitenden Text verfassen.
Angebotstext:
Hier verfassen Sie nun Ihren Angebotstext. In diesem Text können folgende Platzhalter enthalten sein.

###BID_IMG:ALIGN### Hiermit bestimmen Sie die Ausrichtung des Angebotsbildes. ALIGN wird dann ersetzt durch left | center | right

###BID_START:FORMAT### Hiermit bestimmen Sie das Datumsformat des Startdatums. Ersetzen Sie FORMAT mit d/m/y oder y/d/m  (y=Jahr, m= Monat, d=Tag)

###BID_END:FORMAT### Hiermit bestimmen Sie das Datumsformat des Enddatums. Ersetzen Sie FORMAT mit d/m/y oder y/d/m, (y=Jahr, m= Monat, d=Tag)

###START_BID### Mit diesem Platzhalter wird das Startgebot angezeigt

###BID_CURRENT### Mit diesem Platzhalter wird das aktuelle Angebot angezeigt.
 
###BID_FORM### Mit diesem Tag wird das Angebotsformular bestehend aus einem E-mail Feld und dem Angebotsfeld angezeigt.
Formular: Hier werden die Formulareigenschaften definiert.
Da die Felder ###BID_EMAIL### und ###BID_AMOUNT### zwei Eingabefelder sind werden die Platzhalter in einen INPUT Tag eingefügt.

###BID_EMAIL### Dieser Platzhalter ist dafür da das E-mail Feld anzuzeigen.
Beispiel: E-mail:&nbsp;<input name="###BID_EMAIL###" type="text" size="30" value="###BID_EMAIL###" />

###BID_AMOUNT### Mit diesem Platzhalter wird das Feld angezeigt in dem das Angebot eingetragen wird und auch das Mindestgebot.
Beispiel: Ihr Gebot:&nbsp;<input name="###BID_AMOUNT###" type="text" size="15" value="###BID_AMOUNT###" />

<!--FORM_ERROR_START--><!--FORM_ERROR_END--> Zwischen diesen beiden Platzhaltern tragen Sie die Fehlermeldung ein wenn ein Feld nicht richtig ausgefüllt wurde. 
Beispiel:  <!--FORM_ERROR_START-->Fehler: Bitte überprüfen Sie die Eingaben<br /><!--FORM_ERROR_END-->

Zum Abschluß wird noch der HTML send Button angefügt.
Beispiel: <input type="submit" name="Submit" value="Gebot abschicken" />
Nach Senden:
Tragen Sie hier die Bestätigungsmeldung ein die dem Webseitenbesucher anzeigt, dass sein Gebot erfolgreich abgeschickt wurde.
Beispiel: Ihr Angebot wurde erfolgreich gesendet. Sie erhalten nun eine Bestätigungsmail.
Von E-Mail: In diesem Feld wird die Absendeadresse definiert.
Von Name: 
In diesem Feld wird der Absendename definiert.
Verify-Email:
Hier wird die E-mail Vorlage erstellt die der Besucher erhält nachdem er sein Gebot abgegeben hat.

###START_BID### Gibt den Startpreis an

###VERIFY_LINK### Erstellt den Link zum Bestätigen des Angebots.

###DELETE_LINK### Erstellt den Link zum Löschen des Angebots.

###EMAIL### Gibt die E-mail Adresse des Bietenden aus

###BID### Gibt aus welcher Angebotsbetrag geboten wurde.

###BID_START:FORMAT### Hiermit bestimmen Sie das Datumsformat des Startdatums. Ersetzen Sie FORMAT mit d/m/y oder y/d/m  (y=Jahr, m= Monat, d=Tag)

###BID_END:FORMAT### Hiermit bestimmen Sie das Datumsformat des Enddatums. Ersetzen Sie FORMAT mit d/m/y oder y/d/m, (y=Jahr, m= Monat, d=Tag)

###BID_URL### Fügt den Link zum Angebot ein
Verify-Text:
Tragen Sie hier die Bestätigungsmeldung ein die dem Webseitenbesucher anzeigt, dass sein Gebot erfolgreich bestätigt wurde.
Beispiel: Ihr Gebot wurde erfolgreich bestätigt.
Gelöscht:
Tragen Sie hier die Bestätigungsmeldung ein die dem Webseitenbesucher anzeigt, dass sein Gebot erfolgreich gelöscht wurde.
Beispiel: Ihr Gebot wurde gelöscht.
Danach:
Hier können Sie noch einen Text verfassen der nach dem eigentlichen Angebot angezeigt wird.
Bearbeiten:
Hier können Sie eine Übersicht der E-mail Adressen sehen, die mitgeboten haben. 
E-mail Adressen die unterstrichen sind, haben ihr Gebot bestätigt. 
E-mail Adressen die nicht unterstrichen sind, haben ihr Gebot nicht bestätigt.

Durch Klicken auf können Sie den Bieter aus der Liste löschen.
