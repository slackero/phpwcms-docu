phpwcms Installation tutorial

Before you begin...

phpwcms has few requirements some of which are optional. They are listed as follows...
  • Sufficient Webspace
  • PHP 4.2.1 or better
    • (recommended 4.3.x)
  • MySQL
  • GD 1 or GD 2 (Optional)
    • Freetype, for use with the Graphical Text module
  • Image Magick
    • Ghostscript (recommended)



Upload phpwcms

After you have downloaded the latest release from the Project Archive you should unzip the packaged filed then upload it to your webspace using your favorite FTP application. Once uploaded successfully it should be look similar to the image below.





Files & Folder settings

You now need to setup the correct permissons for certain files and folders. This should be done using CHMOD from within your FTP Application.

Folder   phpwcms_tmp   777
Folder   phpwcms_tmp/thumb_list   777
Folder   phpwcms_tmp/thumb_preview   777
Folder   phpwcms_ftp   777
Folder   phpwcms_filestorage/1...10   777
Folder   content/images   777
Folder   content/gt   777
Folder   content/pages   777
         
File   setup/setup.conf.inc.php   666
File   template/inc_css/frontend.css   666
File   include/inc_conf/conf.indexpage.inc.php   666
File   template/inc_default/startup.html   666


Starting the installation



To start the Installation process you open a new Browser window. Enter the address your phpwcms files have been uploaded to including any sub-directories you may have used followed by /setup/index.php
(e.g. http://www.YourDomain.com/YourSubDirs/setup/index.php)

After reading the License agreement click the I AGREE (CONTINUE) Button to accept the terms and conditions underwich to agree to abide. you will now be presented with a screen such as teh image left.



This screen left will confirm to your that you have acepted the licensing agreement and that the installation will commence on clicking the Start install and Setup button provided.


Database Setup



For this step you will need your mySQL database login details (Database name, User name & User password). If you do not yet know your database login details you should ask host for this. You Will only need to use the prepend table names If you intented to install more than one version of phpwcms within the same database. If you are unsure of this the you are advised to leave this field empty.

If this is your first installation of phpWMS and no previous database has been installed you should now check the create all tables instructing the setup process to build the database structure automatically. Leaving this box unchecked is for those users whom are updating or reinstalling phpwcms over an existing version.

Your final task in this step is to click on the send db data any errors within this step are more than likely to be caused and cured by the validaty of data that you have provided and you should check that all is correct before continuing with this setup.



Site information and Webmaster account


fig 1.4 (Click to enlarge)
Step 4 requests that you provide details of your website and for the default Webmaster/Administrator of the site fig 1.4.
The first field is for your full domain address followed by a 'trailing slash' /
The next three fields setup the 'User name', 'User password' and 'e-mail address' for the Webmaster/Administrator. The details you supply in this step are not equivalent to your database details from step 3. It is strongly recommended that you do not use the same information for both steps.

If you had checked the Create all DB tables in the previous step the nyou must chack the create admin account in this step otherwise no admin account would be created thus providing you with no way of loging in.

The remaining fields may be left 'as is' unless you fully understand the settings you wish to use here. You will be able to change and alter the settign for these fields at a later date via edits to the include/conf.inc.php
Click the send site data button to commit the information and proceed to step 5.


Setup Pathes Values


fig 1.5 (Click to enlarge)
Normally you don't need to change here anything.

Only take a look at the field "phpwcms root", wich only should filled out with the subdir where your phpwcms is uploaded.

Hit the "send path values" button to come to the next step.





Content Settings


fig 1.6 (Click to enlarge)
This step (fig 1.6) offers a few configurable setting. It is not nessasery to alter any settigns here unless you fully understand thier implications fully.

In the field  "upload file size" you can setup the maximum file size wich will be allowed to upload. via the filecenter in the backend.

"content width" determine the width of the main content area.

"list image" determine the size of the preview images.

"preview image" determine the size of the images in normal view.

"logout time" determine after wich inactivity you ill be logged out. Standard is 1800 seconds, wich will be 30 min.

"page compression" determine if the page compression will be enabled..

And at last you can setup a the "charset".

Hit the "send content values" button to come to the last step.


create and upload the conf.inc.php


fig 1.7 (Click to enlarge)
So if everything has gone well and there are no permissions or path problems then your screen should resemble fig 1.7

If not, you should now refer to Error messages and correct any errors before continuing.

Once you have a full compliment of green boxes you should now download and save the newly generated config.inc.php file to your include/inc_conf/conf..inc.php replacing the existing file.

Now please delete the Setup folder as this would now presents a security risk to your website.

Once completed you have successfully completed you phpwcms instalation.
You may nowLOGIN via link in the upper right of your screen to the backend with the username and password presented in step 4.
