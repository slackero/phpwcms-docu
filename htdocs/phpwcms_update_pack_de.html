phpwcms Update Pack

HINWEIS

Ab sofort werden auf der Docu Seite keine CVS Versionen von phpwcms mehr bereitgestellt, da es kein öffentliches CVS mehr gibt. Dafür finden Sie ab sofort hier immer Update Packs, die nur die Änderungen auf die aktuelle Version beinhalten.

Bitte beachten Sie dabei, dass dies dann grundsätzlich ein Update ist und Sie somit darauf achten müssen, die Dateien an Ihr System anzupassen!

Dies trifft besonders für Dateien in den Ordnern

config/phpwcms/conf.inc.php
config/phpwcms/conf.template_default.inc.php
template/inc_css/frontend.css

sowie alle anderen Dateien zu, die Sie angepasst haben.

Die Update Packs finden Sie hier in 2 Varianten. Zum einen eine Version, die ein Update von RC 1.1 vom 27.08.2005 auf die aktuelle Version möglich macht (BITTE beachten Sie hierbei folgende Update Anweisungen RC 1.1 auf 1.2.x)  und eine Version, die sich nur auf die Änderungen zur vorherigen Version bezieht. Sollten Sie nicht wissen, welche phpwcms Version sie zum Schluß installiert hatten, updaten Sie bitte Ihr phpwcms erst einmal komplett. Danach können Sie dann die kleinen Updates benutzen und so Ihr phpwcms immer auf dem neuesten Stand halten.

Nähere Informationen zu den Updates finden Sie in folgendem Forum Beitrag

http://forum.phpwcms.org/viewtopic.php?t=8390
