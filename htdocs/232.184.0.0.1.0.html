File Administration
Administer and add files


click to enlarge

It is very easy to upload pictures, files and documents with the built-in data management of phpwcms. These files can be used in different articles. The data management creates a windows-like folder structure. You can easily create folders and unblock them for all editors or just for yourself. It is easy to upload files from your hard drive.

Standard documents like Microsoft Word or PDF will be shown with the well known standard icons. Pictures in all current formats will be shown by a thumbnail.
Try it; the data-manager is really easy to use.
