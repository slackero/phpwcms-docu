Content Part "Sitemap"

content part "sitemap"

With this content part you can offer your visitors a sitemap.

Field descriptions

before: Here you enter the text that appears before the sitemap
after: Here you enter the text that appears after the sitemap

level icon:

 Here can you choose your own image for the levels e.g.  img/article/fill_in_here.gif  for
article icon: Here you can choose your own image for the articles e.g. 
img/usricon/usr_admin.gif for
start at:
You can decide from which structure level the sitemap starts to show by choosing a level. The standard start structure level is Home
display:
Here you select from one of the following options:
1. structure levels only: the generated sitemap will only display the structure levels
2. structure levels + articles: the generated sitemap will display both structure levels and articles 
level class: Create a class in the frontend.css for the levels and type only the class name in this field.
article class: Create a class in the frontend.css for the article view and type only the class name into this field.
counter: Here you select from one of the following options:
1. don’t add to class name: (no data available)
2. add to class name: (no data available)
