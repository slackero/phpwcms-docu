Content Part "Sitemap"

Content Part "Sitemap"

Mit diesem Content Part ist es möglich, eine sogenannte Sitemap Ihren Besuchern zur Verfügung zu stellen.

 

Feldbeschreibungen

Davor: Hier können Sie den Text eingeben, der vor der Sitemap erscheinen soll.
Danach: Hier können Sie den Text eingeben, der nach der Sitemap erscheinen soll.

Ebene Icon:

 Hier können Sie Ihr eigenes Bild für die Ebenen auswählen, z.B. img/article/fill_in_here.gif  für
Artikel Icon: Hier können Sie Ihr eigenes Bild für die Artikel auswählen, z.B. img/usricon/usr_admin.gif für
Startet ab:
Durch auswählen einer ebene können Sie bestimmen von welcher Strukturebene aus die Sitemap angezeigt wird. Standard start Strukturebene ist Home
Anzeigen:
Hier können Sie eine der folgenden Optionen auswählen:
1. Nur Seitenebenen: Die erzeugte Sitemap zeigt nur die Seitenebenen an.
2. Seitenebenen und Artikel: Die erzeugte Sitemap zeigt sowohl Seitenebenen als auch Artikel an. 
Ebene Class: Erzeugen Sie eine Klasse für die Seitenebenen in der frontend.css und geben Sie nur den Klassennamen in dieses Feld ein.
Artikel Class: Erzeugen Sie eine Klasse für die Artikelansicht in der frontend.css und geben Sie nur den Klassennamen in dieses Feld ein.
Zähler: Hier können Sie eine der folgenden Optionen auswählen:
1. nicht im Class-Name mitführen: (Keine Daten verfügbar)
2. im Class-Name mitführen: (Keine Daten verfügbar)
