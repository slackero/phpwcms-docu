Add Ons

Hier finden Sie Erweiterungen für phpWCMS, die KEINE Änderungen am Quelltext benötigen. Manche Erweiterungen müssen duch sogenannte Replacement Tags ausgeführt werden. Andere arbeiten einfach im Hintergrund ohne das etwas in einem Template zugefügt werden muß. Wir haben Sie an dieser Stelle aufgeteilt in Replacement Tags und System Add Ons.

Eine große Anzahl von Erweiterungen finden Sie im Ordner template/inc_script/frontend_render/disabled. Verschieben Sie die gewünschte Erweiterung in den Ordner frontend_render um diese nutzen zu können.

Wie Sie im inc_script Verzeichnis sehen können gibt es dort 2 Ordner einen frontend_render und frontend_init. Der Unterschied zwischen den beiden ist, das alles was im frontend_render liegt beim Seitebaufbau ausgeführt wird. Im gegensatz zu frontend_init die scripte die man dort ausführen kann werden ausgeführt vor dem erstellen der Seite.

Um Add Ons zu nutzen muss in der conf.inc.php folgende Änderung vorgenommen werden:

Beide der folgenden Werte müssen dazu auf 1 gesetzt werden.

$phpwcms["allow_ext_init"]    = 1; 
$phpwcms["allow_ext_render"]  = 1;
