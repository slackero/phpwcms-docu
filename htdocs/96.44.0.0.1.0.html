Replacement Tags für Grafiken

HINWEIS
Alle Image Tags beziehen sich auf das Verzeichnis "Picture". Wenn Sie aus einem anderen Ordner Bilder mit dem Image Tag aufrufen wollen muß der Link z.b. so aussehen {IMAGE:../content/images/image.jpg}

{IMAGE:Imagename}
Verwendung: Erzeugt ein <img>-Tag mit der angegebenen Bilddatei.
Schreibweise: {IMAGE:icon/DateiDownload.jpg}
Anzeige:
Bemerkung:

Das Verzeichnis icon liegt unmittelbar unterhalb von picture.
Der komplette Pfadname zur Bilddatei ist in diesem Fall:

www.meine-domain.de/picture/icon/DateiDownload.jpg

 

{IMAGE_LEFT:imagename}
Verwendung: Erzeugt ein <img>-Tag mit der angegebenen Bilddatei.
Das Bild wird linksbündig ausgerichtet.
Schreibweise: {IMAGE_LEFT:Jaguar.jpg}
Anzeige:
Bemerkung:

Das Bild Jaguar.jpg liegt im Verzeichnis picture.
Der komplette Pfadname zur Bilddatei ist in diesem Fall:

www.meine-domain.de/picture/Jaguar.jpg

 

{IMAGE_RIGHT:Imagename}
Verwendung: Erzeugt ein <img>-Tag mit der angegebenen Bilddatei.
Das Bild wird rechtsbündig ausgerichtet.
Schreibweise: {IMAGE_RIGHT:Blatt.jpg}
Anzeige:
Bemerkung:

Das Bild Blatt.jpg liegt im Verzeichnis picture.
Der komplette Pfadname zur Bilddatei ist in diesem Fall:

www.meine-domain.de/picture/Blatt.jpg

 

{RANDOM:Image_Pfad}
Verwendung: Erzeugt ein <img>-Tag für Zufallsbilder aus dem angegebenen Verzeichnis. 
Bei jedem Seitenaufruf wird ein anders Zufallsbild angezeigt.
Schreibweise: {RANDOM:picture/random_175}
Anzeige:
Bemerkung:

Das Verzeichnis random_175 ist ein Unterverzeichnis von picture.
Der komplette Pfadname zu den Random-Bildern ist:

www.meine-domain.de/picture/random_175/

Der Bildvorrat des Random-Ordners sollte nur Bilder mit der gleichen Abmessung beinhalten! 
Die Bilder werden unmittelbar aus dem Verzeichnis geladen und nicht neu vom Programm berechnet.
 

{SPACER:px-Breite x px-Hoehe}
Verwendung: Ausgabe eines grafischen Abstandselements zum Erzeugen einer "freien Fläche". Eingabe von Breite und Höhe in Pixeln.
Schreibweise: {SPACER:250x5}


Beispiel:

ANFANG -|{SPACER:250x5}|- ENDE
Anzeige: ANFANG -||- ENDE 
Bemerkung: Der Zwischenraum zwischen den beiden Marken -| und |- wurde hier mit exakt 250 Pixeln festgelegt.
