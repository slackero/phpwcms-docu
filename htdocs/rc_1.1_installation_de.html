phpwcms Installations Anleitung

Bevor Sie beginnen

Welche Voraussetzungen müssen gegeben sein, um mit phpwcms arbeiten zu können?
  1. Webspace
  2. PHP ab der Version 4.2.1 besser jedoch ab 4.3.4
  3. MySQL
  4. GD 1 oder GD2 (Optional)
  5. Image Magick / Ghostscript (Empfohlen)

Hochladen von phpwcms

Nachdem Sie phpwcms von der Projektseite heruntergeladen und entpackt haben, können Sie alles mit einem FTP-Programm auf Ihren Webspace hochladen. Nachdem Sie es hochgeladen haben, sollte der Inhalt des Ordners so aussehen wie auf der Grafik.

Setzen der Dateirechte

Sie müssen nun den folgenden Dateien und Ordnern, die zum Funktionieren von PHPWCMS notwendig sind, die nötigen Rechte geben. Diese können Sie mit Ihrem FTP-Programm setzen. 


Verzeichnis   phpwcms_tmp   777
Verzeichnis   phpwcms_tmp/thumb_list   777
Verzeichnis   phpwcms_tmp/thumb_preview   777
Verzeichnis   phpwcms_ftp   777
Verzeichnis   phpwcms_template   777
Verzeichnis   phpwcms_filestorage/1...10   777
Verzeichnis   content/images   777
Verzeichnis   content/gt   777
         
Datei   setup/setup.conf.inc.php   666
Datei   template/inc_css/frontend.css   666
Datei   include/inc_conf/conf.indexpage.inc.php   666
Datei   template/inc_default/startup.html   666

Aufrufen der Installation

Nun rufen Sie in Ihrem Browser das Installationsscript auf, dessen Pfad wie folgt lautet:
http://www.ihre-domain.tld/setup/index.php

Hierbei gehen wir davon aus, dass Sie phpwcms im Root (Hauptverzeichnis) Ihres Webspaces hochgeladen haben.
Falls das nicht der Fall ist, muß der Pfad so aussehen:

http://www.ihre-domain.tld/unterordner/setup/index.php

Nach dem Durchlesen der Lizenzbestimmung drücken Sie einfach auf die Schaltfläche
"I AGREE (CONTINUE)". So gelangen Sie zum nächsten Schritt.

Hier werden Sie nochmal aufgefordert zu überprüfen, ob auch alle notwendigen Dinge auf Ihrem Webspace vorhanden sind.

Drücken Sie dann auf "Start Install and Setup".






Datenbank Setup

Für diesen Schritt benötigen Sie Ihre Zugangsdaten zur MySql-Datenbank.
Diese Daten bekommen Sie von Ihrem Webhoster.

Das Feld "prepend table names" müssen Sie nur dann ausfüllen, wenn Sie mehrere phpwcms-Installationen auf einer Datenbank durchführen wollen.

Bitte setzen Sie auch einen Haken bei "create all tables". Damit werden ALLE benötigten Tabellen für phpwcms in der Datenbank angelegt.

Klicken Sie dann auf "send db data".

Sollten Sie jetzt - bei einer Erstinstallation - eine Fehlermeldung bekommen, überprüfen Sie bitte Ihre Eingaben.

Seiteninformationen und Admin Account

Hier wird jetzt die URL Ihrer phpwcms-Installation bestimmt. Diese wird vom Setup-Script ausgelesen. Achten Sie darauf, dass dort nur der Domainname steht, der mit einem /. endet.

Danach vergeben Sie den Namen, Passwort und E-mail Adresse für den Administrator.
Mit dem Namen und dem Passwort können Sie sich später in Ihrem phpwcms anmelden  und den Inhalt für Ihre Seite erstellen. Vergessen Sie aber auf keinem Fall den Haken bei "create admin account" zu setzen, da sonst kein Administrator-Konto erstellt wird.

Die anderen Felder können Sie erstmal ungeachtet lassen. Im Bedarfsfall können diese später in der Datei "conf.inc.php" im Ordner "include/inc_conf" eingetragen werden.

Durch Anklicken des "send site data" Button gelangen Sie zum nächsten Schritt.

Interne Pfade

Hier brauchen Sie nichts zu verändern.
Achten Sie nur darauf, das bei einer Installation von phpwcms
in einem Unterordner nur der Name des Unterordners in dem Feld
"phpwcms root" steht.


Klicken Sie dann auf "send path values".








Inhaltseinstellungen

 Hier können Sie nun einige Einstellungen vornehmen - wobei es jedoch eigentlich erstmal nicht erforderlich ist.

Im Feld "upload file size" können Sie die maximale Größe der Datei bestimmen,
die unter Dateien hochgeladen werden darf.

"content width" bestimmt die Breite des Hauptinhaltsbereiches.

"list image" bestimmt die Größe der Vorschaugrafiken.

"preview image" bestimmt die Größe der Bilder in der Normalansicht.

"logout time" bestimmt, nach welcher Zeit, in der man inaktiv ist, ein "Ausloggen" stattfindet. Standard ist 1800 Sekunden, sprich 30 min.

"page compression" bestimmt ob die Seitenkomprimierung eingeschaltet sein soll.

Zu guter Letzt können Sie unter "charset" noch den Zeichensatz bestimmen.

Klicken Sie nun auf "send content values" um zum letzten Schritt zu gelangen.

conf.inc.php erstellen und hochladen

 So wenn jetzt alles glatt gegangen ist und kein Pfad- oder Rechteproblem vorliegt, sollten Sie jetzt alle Balken so wie in der Grafik angezeigt, in grün sehen. 

Falls nicht, finden Sie im nächsten Bereich alle Fehlermeldungen und wie sie behoben werden können.

Nichts desto trotz klicken Sie jetzt auf "Download the created config file" und speichern Sie es auf Ihrer Festplatte ab. Ersetzen Sie dann die Datei "conf.inc.php" im Ordner "include/inc_conf/" mit Ihrer heruntergeladenen Version. 

Löschen Sie dann bitte aus Sicherheitsgründen den kompletten Setup Ordner. Wenn Sie das nicht machen, stellt das ein großes Sicherheitsrisiko dar.

Danach können Sie oben rechts auf den "LOGIN" drücken und sich mit Ihrem Usernamen und Passwort anmelden.
