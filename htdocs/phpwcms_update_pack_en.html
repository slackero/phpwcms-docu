phpwcms update pack

Attention

Starting now no CVS versions of phpwcms will be provided on this docu site because no more public CVS is going to be published by the author. Instead you will always find update packs which will only contain changes to the up-to-date version.

This relates in the first place to the following files and folders:

config/phpwcms/conf.inc.php
config/phpwcms/conf.template_default.inc.php
template/inc_css/frontend.css

As well as all other files you adjusted to your specific needs.
You will find the update packs in 2 versions. First a version which makes the update possible from RC 1.1 from Aug. 27, 2005 to the up-to-date version (PLEASE note the update instructions from RC 1.1 to 1.2.x!) and a version that only contains changes from the proceeding version. In case you don’t know which phpwcms version you have had installed at last please run a complete update which means download and install the latest phpwcms pack that you will find in the news.

Changes in the CVS will be described in the forum, you will find the thread here

http://www.phpwcms.de/forum/viewtopic.php?t=8390
