Chat

Chat für die Redakteure

Der Backend-Chat dient dazu Diskussionen direkt mit mehreren Redakteuren oder Admins zu führen. Wenn man nur einer Person eine Nachricht hinterlassen möchte, kann man dies über die Nachrichten-Funktion tun. Man kann den Chat auch als eine Art Forum verwenden und allen andern Redakteuren dort Nachrichten hinterlassen, worauf diese direkt antworten können. Dies wird dadurch möglich, dass der Verlauf der Nachrichten in der Datenbank gespeichert wird und jederzeit nachvollzogen werden kann.

 

Das Chatfenster

Bild 1 - Chateingangseite
Bild 2 - Das Chatfenster

Um zum Chatfenster zu gelangen klicken Sie auf
"Gehe zum Chat (hier  klicken)" (Bild 1)

(Bild 2) Schreiben Sie Ihre Nachricht in das Feld vor dem Button "Send", zum Senden der Nachricht drücken Sie entweder "Enter" oder auf die Schaltfläche .

Im rechten oberen Bereich sehen "Anzeige Chat Verlauf". Dort können Sie bestimmen wieviele Zeilen des Chats angezeigt werden.
