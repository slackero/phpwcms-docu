Nachricht lesen
und beantworten


Wenn Sie eine Interne Nachricht erhalten haben sehen Sie das sofort nach dem Einloggen in das System oben rechts im roten Balken sehen Sie dieses


Symbol. Jetzt haben Sie 2 Möglichkeiten die Nachricht auzurufen. Entweder Sie klicken das Symbol an oder Sie klicken in der gleichen Zeile auf "Nachricht"

Nachrichten Überblick



Nachdem Sie nun auf eine der beiden möglichkeiten geklickt haben. Gelangen Sie zur Nachrichten Übersicht. Dort können sehen Sie nun wer Ihnen eine Nachricht und mit welchem Betreff gesendet hat. Am Ende der Zeile sehen Sie 2 Icons.

Das erste Icon leitet Sie zur Antwortseite weiter.
Das zweite Icon löscht die Nachricht. Gelöschte Nachrichten werden aber nicht sofort unwiederbringlich gelöscht, sondern werden in dern Ordner "Gelöscht" verschoben.

Wenn Sie jetzt entweder auf auf den Absender der Nachricht oder den Betreff klicken können Sie die Nachricht lesen.


Nachricht lesen



Hier können Sie nun die Ihre Nachricht Lesen. Unter der Nachricht sehen Sie 4 Schaltflächen.

Das sind die Bedeutungen der einzelnen Schaltflächen
 
schließen Sie die Nachricht
erstellen einer neuen Nachricht ohne auf die Nachricht die Sie gerade lesen bezug zu nehmen.
gelangen Sie zur Antwortseite
    löscht die Nachricht.

Nachricht beantworten



Hier können Sie nun die Nachricht beantworten. Es funktioniert genauso wie wenn Sie eine Nachricht verfassen. mIt der einzigen Änderung das Sie hier keinen Empfänger auswählen müssen.
