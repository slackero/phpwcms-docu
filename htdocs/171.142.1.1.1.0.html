Content Part "Ext. Content"

With this content part it's possible to view your own created HTML in a phpwcms installation.

source: Here you can choose from one of the following options: 

1. file from structure:  If you want use html files that you have uploaded to the content/pages folder choose this option
2. custom path/file or URL: Use this option if the page you want to view is outside of your phpwcms installation.



file/URL:  If you choose the custom path/file or URL option you can insert here the whole path to the file you want to display. e.g. http://your_domain.com/file.html

select file: You can upload your own html files into the content/pages folder. You can also create the folders. All folders / files you will see in the select file field below. You can choose them by setting the dot in front of the file.
