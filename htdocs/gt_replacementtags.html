Replacement Tags

Replacement Tags

Now we have fonts, color and styles defined, but how can we use this in
phpWCS? 

For that, the GT MOD has integrated a replacement tag with which it is
really easy to handle.

{GT:stil}Mein Text{/GT}

Usage:   applies a in the backend defined style on every arbitrary text.
Notation: {GT:Stil} my text{/GT}
shown as:
Remark: The defined style will be applied to the text. The tag can also include another text-tag, e.g., Author: {GT:Stil2} {AUTHOR} {/GT}
shown as:

for the attention of all „copy & paste“ fans!

To show the Replacement tags in this documentation, without being
immediately interpreted, we had to insert in „title“ and „Notation“ after
the left parenthesis a SPACER, which is of course not allowed in a real RT!
( <img src="img/leer.gif" border="0" width="1" height="1" alt="" /> ) 

So if you copy the RT from this manual, the function is not guaranteed!
We really advice to insert the Replacement-tags by hand in the code or to
remove the SPACER!
