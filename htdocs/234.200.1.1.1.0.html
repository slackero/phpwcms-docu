Upload Files

Single File

click to enlarge
Only if you created a directory, you can upload files.

To upload a file in a directory you have to click on the [ IMG ] icon of the
wished directory. In the following dialog you can make next informations:

Folder: here you can choose the directory you take.
Upload: Click on „choose files“ to take a file on your hard drive,
wich you wanna upload.

Keywords (optional): here you can type in additional keywords, with them you can
search later.
Info (optinal): Here you can fill in additional Informations.
Status: decide if the file is:

  1. active (means that the file can be used in articles)
  2. public (means that other users can use this file)

Upload file: With pushing on that button you confirm the procedure. Depending on
the file size and connection to the internet it could take a while to upload the
files. If the new file is a picture, it will be automaticly created a preview.

Several Files

click to enlarge
To upload several files, phpwcms offers a smart possibility of the FTP-Takeover. You need for that a external program, user name and password, to log in on the FTP-server. With that you upload the files in the setted folder by the installation of phpwcms (standard: phpwcms_ftp). Follow this rules to upload the files:
  1. Upload the wished files per FTP-Program in the Directory “phpwcms_ftp” or in the on the installation defined directory.
  2. Click in the Pphpwcms on „ftp take over“ in the left navigation. You see a overview of the uploaded files.
  1. Select one or more files in the column “mark”. Hint: you can select all by
    clicking on “ALL”
  2. Choose now under „directory“, the directory where you wanna upload the files in.
  3. Give as by uploading of single files a optional keyword and/or a discription.
  4. Choose a status of the files like by uploading of single files.
  5. Choose if a preview of the images should be created automaticly.
  6. Click on „take over selected files” to finish the process. Depending on the
    file size and connection to the internet it could take a while to upload the
    files.
  7. Click on „file center“ in the left navigaten to get back into the file overview:
