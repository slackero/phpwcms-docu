Benutzer Anlegen

Nachdem Sie auf geklickt haben sehen Sie nun folgende Eingabemaske.
Legende
Felder Beschreibung

Benutzer:

 Tragen Sie hier den Anmeldenamen des neuen Benutzers ein. Standardmäßig wird aber von System ein Name vorgeschlagen.

Passwort:

 Hier bestimmen Sie das Passwort des Benutzers. Dieses kann der Benutzer aber später unter Profil ändern.
Hier wird ebenfalls ein Passwort vom System selber vorgeschlagen.
E-Mail: Tragen Sie hier die E-mail Adresse des neuen Benutzers ein.
Aktivieren: Hier müssen Sie den Haken setzen da sich sonst der Benutzer nicht anmelden kann.

Administrator:

 Wenn Sie hier einen Haken setzen geben Sie dem neuen Benutzer Adminstratorrechte.

Überprüfung:

 Wenn Sie hier den Haken setzen erhält der neue Benutzer nach Drücken der Schaltfläche "Benutzerdaten senden" eine E-Mail mit seinen Anmeldedaten.

Benutzer anlegen abschließen

Nachdem Sie nun alle Felder ausgefüllt haben, drücken Sie auf die Schaltfläche


und die von Ihnen vorgenommenen Benutzereinstellungen werden in der Datenbank gespeichert.
