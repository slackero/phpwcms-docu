Standard CSS

Die Standard CSS "frontend.css" zeichnet sich für das Frontend verantwortlich und findet sich nach einer "frischen" Installation im Verzeichnis /template/inc_css wieder. Es ist zu beachten, dass der CHMOD auf 666 gesetzt ist. Andernfalls können Änderungen innerhalb des Backends von phpWcms nicht gespeichert werden.

Da die  frontend.css bei weitem nicht alle möglichen, in phpWcms vorhanden CSS classes beinhaltet, wird der Einsatz einer zusätzlichen CSS-Datei empfohlen, die ebenfalls im Verzeichnis /template/inc_css abgelegt werden sollte. 

Auf diese Weise können Sie Styles hinzufügen und haben bei möglichen Upgrades von phpWcms weniger Probleme.
