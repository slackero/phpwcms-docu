Content Part "Ext. Content"

Content Part "Externer Content"

Mit diesem Content Part ist es möglich eigene html Seiten in phpwcms einzubinden.

Herkunft: Hier können Sie zwischen folgende Optionen wählen: 

1. Datei aus der Struktur:  wählen Sie diese Option wenn Sie Dateien in den Ordner content/pages hochgeladen haben.

2. Eigene Pfad / Datei oder URL: wählen Sie diese Option wenn Sie auf eine html Datei ausserhalb des content/pages Ordners zugreifen wollen.

 

DATEI / URL: Wenn sie die Option Eigene Pfad / Datei oder URL gewählt haben geben Sie hier den Pfad zu der html datei an die Sie aufrufen wollen (z.b. http://www.ihre_domain.de/seite.html)

Datei wählen : Sie können im Order content/pages auch ordner erstellen, die Dateien die Sie dort hochladen werden wie auf dem Bild links angezeigt. Durch setzten der Markierung (Punkt) können Sie eine Datei auswählen.
