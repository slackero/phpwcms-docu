Content Part "FAQ"

Frequently Asked Questions („FAQ“ aus dem englischen für häufig gestellte Fragen) sind eine Zusammenstellung von häufig gestellten Fragen und den dazugehörigen Antworten zu einem Thema. Sie können dazu beitragen, den Posteingang des Webmasters ein wenig zu entlasten und werden oftmals vorbeugend angelegt.

phpWcms bietet anhand des CP FAQ die Möglichkeit, selbst eine solche Liste aufzustellen. Wie man dazu vorgeht möchten wir hier kurz erläutern. Da die eigentliche FAQ denkbar einfach ist, haben wir uns gemeinsam mit Oliver Georgi erlaubt eine Variante mit JavaScript zu erstellen.

Und so wird's gemacht:

Vorlage:
Im Verzeichnis /template/inc_cntpart/faq finden Sie die Datei default.tmpl. Anhand dieser Basis können Sie sich Ihr eigenes Template erstellen. Es wird dabei empfohlen die default.tmpl nicht zu überschreiben.

Da wir das ganze ein wenig beschleunigen möchten, haben wir an dieser Stelle bereits ein Template angefertig. Dieses Template wird in zukünftigen Versionen von phpWcms im Verzeichnis /template/inc_cntpart/faq  verfügbar sein. Wenn Sie jedoch bereits die Ungeduld packt, oder eine ältere Version des CMS im Einsatz haben und kein Upgrade vornehmen möchten, können sie den nachfolgenden Quelltext am Seitenende übernehmen und als Datei "ihrname.tmpl" speichern und in das Verzeichnis /template/inc_cntpart/faq auf Ihrem Webserver übertragen. Wählen Sie anschließend diese Vorlage aus.

Frage:
Hier definieren Sie eine Frage, von der Sie denken, dass Sie von den Usern Ihrer Website oft gestellt wird bzw. gestellt werden könnte.

Antwort:
Passend zur Frage sollte an dieser Stelle die Auflösung erfolgen.

Bild:
An dieser Stelle können Sie ein Bild aus dem Dateimanager auswählen.

Hinweis:
Wenn Sie das Aufklapp FAQ verwenden möchten, sollten Sie Überschriften innerhalb dieses CP's vermeiden, da die Aufklappfunktion sonst nicht greift.

Quelltext Template:

<div class="docuFAQ">
 [TITLE]<h3>{TITLE}</h3>[/TITLE][SUBTITLE]
 <h4>{SUBTITLE}</h4>[/SUBTITLE]
 
    <h5 id="question-{FAQ_ID}" onclick="toggleLayerDisplay('faq-{FAQ_ID}', 'block'); toggleLayerDisplay('question-{FAQ_ID}', 'none');">
    {FAQ_QUESTION}
    </h5>
   
    [FAQ_ANSWER]
    <div class="faqEntry" id="faq-{FAQ_ID}">
        <h5 onclick="toggleLayerDisplay('faq-{FAQ_ID}', 'none'); toggleLayerDisplay('question-{FAQ_ID}', 'block');">
        {FAQ_QUESTION}
        </h5>
       
        [FAQ_IMAGE]
        <div class="faqImage">{FAQ_IMAGE}[FAQ_CAPTION]<em class="faqCaption">{FAQ_CAPTION}</em>[/FAQ_CAPTION]</div>
        [/FAQ_IMAGE]
       
        {FAQ_ANSWER}
    </div>
    [/FAQ_ANSWER]

</div>

Kopieren Sie anschließend den nachfolgenden Quellcode in Ihre frontend.css:

    padding:0px;
}

    background-position:left 3px;
    background-repeat:no-repeat;
    cursor:pointer;
}

    background-repeat:no-repeat;
}

    float:right;

    margin:3px 3px 15px 15px;
}

.docuFAQ .faqCaption {
    padding:0px;
    margin:4px 0 0 0;
    font-size:9px;
}

.docuFAQ p {
    padding:0 0 0 20px;
    margin:0px;
}

.docuFAQ .faqEntry {
    border-bottom:1px dotted #dae7f6;
    margin:0 0 15px 0;
    padding:0 0 10px 0;
    display:none;
}
