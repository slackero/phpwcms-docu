Replacement Tag

Nachdem Sie Schriftarten, Farben und Stile definiert haben, können Sie den "GT-MOD" mit Hilfe eines Replacement Tags nutzen. Dadurch könen Sie relativ einfach die von Ihnen erstellten Stile einsetzen.

RT: GT-MOD
Verwendung: Wendet einen im Backend definierten Stil auf einen beliebigen Text an
Schreibweise: {GT:Stil} Mein Text {/GT}
Anzeige:
Bemerkung: Der definierte Stil wird auf den Text angewandt. Der Tag kann auch als Text einen weiteren Tag enthalten, z.B. Author :
{GT:Stil2} {AUTHOR} {/GT}
