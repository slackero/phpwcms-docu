Content Part "Einfacher Text"

Einfacher Text:
Fügen Sie hier den von Ihnen gewünschten Text ein. Sie können hier mehrere Absätze hintereinander einfügen. Sollten Sie allerdings Zwischenüberschriften im Text benötigen, sollten Sie mehrere Inhaltsbausteine „Einfacher Text“ hintereinander in den Artikel einfügen und jeweils die Felder „Inhaltsüberschrift“ und/oder „Untertitel“ belegen. Auszeichnungen im Text können Sie durch die nachfolgenden Codes vornehmen.
 
Um das eingeschlossene Wort fett darzustellen:
[b]Text[/b]

Um das eingeschlossene Wort kursiv darzustellen:
[i]Text[/i]

Um das eingeschlossene Wort zu unterstreichen: 
[u]Text[/u]
 

Content Part "Einfacher Text"

Seit der Version 1.3.3 gibt es eine zusätzliches Auswahlfeld.

Unter Vorlage können Sie Vorlagen für den Content Part "Einfacher Text" erstellen.

Eine Standardvorlage finden sie wenn ...

  1.  ... Sie eine neu installiert Version 1.3.3 nutzen unter:
    template/inc_cntpart/plaintext

  2. .. Sie eine Upgedatete Version nutzen unter:
    template/inc_cntpart/plaintext


Vorlagen können nur von Personen verändert werden, die über einen  Zugang zum Server / Webspace verfügen.

Die Standardvorlage sieht im Quelltext wie folgt aus:

[TITLE]<h4>{TITLE}</h4>[/TITLE]
[SUBTITLE]<h5>{SUBTITLE}</h5>[/SUBTITLE]
[TEXT]<p>{TEXT}</p>[/TEXT]

Hinweis zur Vorlage / Template:
Dieser Content Part hat eine eigene Vorlage (Template). Es wird empfohlen wenn Sie die Standardvorlage abändern diese auch umzubenennen und die umbenannte Vorlage zu verwenden. Da Sie somit kein Problem bei einem Update haben. Und Ihre Vorlage nicht überschrieben wird.
