Content Part "Link List"



Link list (optional): Enter each link using the following format: 

designation|address 

Enter each link on a new line. If you enter an e-mail address link, precede the e-mail address with "mailto:" , for example, "mailto:info@mydomain.com".





A sample link list might look like this:

contact me|mailto:Info@mydomain.com
my homepage|http://www.mydomain.com 

Using this sample list, a visitor to the site would see the following: 

contact me
my homepage

To open Links in an external browser window the code would look like this

my homepage|http://www.mydomain.com _blank
