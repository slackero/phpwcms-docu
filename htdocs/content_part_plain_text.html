Content Part "Plain Text"

Content Part "plain text"

plain text (optional): Insert the text you wish to add here. You can insert several paragraphs one after another here. However, if you should need headings for other than the first paragraph, you should insert multiple “plain text” content-parts one after another into the article and in each case, use the fields "content title" and/or "sub-title". 

Special formatting in the text can be done by using the following codes:
[b]Text[/b] to make the enclosed text appear in bold-face.
[i]Text[/i] to make the enclosed text italicized.
[u]Text[/u] to make the enclosed text underlined.

Since version 1.3.3 there is an additional selection field.

Under template you can create templates for the content part "Plain text". With this possibility  you can create several templates for different parts.

You can find a standard template under

a. If you use e newly installed version 1.3.3:
     template/inc_cntpart/plaintext

b. If you use an upgraded version:
     template/inc_cntpart/plaintext

Only a person with access to the server/webspace can change the templates.

The source code of the standard template looks like this:

[TITLE]<h4>{TITLE}</h4>[/TITLE]
[SUBTITLE]<h5>{SUBTITLE}</h5>[/SUBTITLE]
[TEXT]<p>{TEXT}</p>[/TEXT]

Notice for Templates:
This content part has an own template. It is recommended when you change the standard template to rename it and to use the renamed template. Then you do not have any problems with an update and your template will not be overwritten.
