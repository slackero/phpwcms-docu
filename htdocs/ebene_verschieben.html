Ebene / Struktur verschieben

Um eine Ebene zu verschieben gehen Sie wie folgt vor.

Durch Klicken auf wählen Sie eine Ebene/Struktur zum verschieben aus. Um den Vorgang abzubrechen Klicken Sie erneut auf das Icon in der ausgewählten Ebene.
 

Dort wo es nun möglich ist die Ebene einzufügen erscheint dieses Icon . 

Durch Klicken auf auf das Icon fügen Sie die Ebene (inkl. Artiklen) an Ihrem neuen Ort ein. Die Aliase werden dabei natürlich auch übernommen. Um den Vorgang abzuschließen drücken Sie auf das Icon Ihrer verschobenen Ebene / Struktur.
