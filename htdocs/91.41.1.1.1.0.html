System Replacement Tags

[PHP] mein_PHP_code ; mehr_PHP_code [/PHP]

Verwendung: Hiermit kann benötigter PHP-Code direkt eingegeben werden
Schreibweise: [PHP]mein_PHP_code[/PHP]
Anzeige: -
Bemerkung: Eignet sich insbesondere für kurze PHP-Anweisungen
 

{BREADCRUMB}

Verwendung: Erzeugt ein sog. Brotkrümel-Menü, welches dem Besucher zeigt, an welcher Stelle der Navigation er sich befindet. 
Ermöglicht ein gezieltes, schritt weises Zurückgehen.
Schreibweise: {BREADCRUMB}
Anzeige: Home > Deutsch > Die Docu > Admin > Replacement-Tags > System Tags
Bemerkung: Sinnvoll bei Sites mit einer tiefen Navigations-Struktur.
Wird normalerweise in der Vorlage eingesetzt.
 

{BREADCRUMB:ID}

Verwendung:   Erzeugt ein sog. Brotkrümel-Menü, welches dem Besucher zeigt, an welcher Stelle der Navigation er sich befindet. 
Ermöglicht ein gezieltes, schritt weises Zurückgehen. Durch vergabe einer ID kann die Startebene bestimmt werden.
Schreibweise: {BREADCRUMB:1}
Anzeige: Deutsch > Die Docu > Admin > Replacement-Tags > System Tags
Bemerkung: Sinnvoll bei Sites mit einer tiefen Navigations-Struktur.
Wird normalerweise in der Vorlage eingesetzt.
 
 

{CONTENT}

Verwendung: Verantwortlich für die Wiedegabe aller Inhalts-Elemente.
Kann nur einmal vergeben werden! 
Steht sinnvoller weise innerhalb der Vorlage im Feld Haupt.
Schreibweise: {CONTENT}
Anzeige: -
Bemerkung: Sollte keine Vorlage eingegeben werden, ist diese RT
im Haupt-Feld vorgegeben!
 

{PHP:mein_externe_PHP_Datei.php}

Verwendung: Hiermit wird ein komplettes PHP-Script eingebunden.
Schreibweise: {PHP:meine_externe_PHP_Datei.php}
Anzeige: -
Bemerkung: Es ist der Dateiname einschließlich der notwendigen Pfadangabe einzugeben!
Längere PHP-Scripte sollten ggf. innerhalb eines iframe in phpWCMS eingebunden werden.
 

{URL:url_der_externen_site}

Verwendung: Hierdurch wird eine externe Internetseite eingebunden.
Schreibweise: {URL:http://www.phpwcms-docu.de}
Anzeige: -
Bemerkung: Einschränkung: 
Nur der Bereich zwischen <BODY> und </BODY> wird übernommen!
 

[BACK] Text oder Bild [/BACK]

Verwendung: Erzeugt eine Zurück-Funktion. 
Bei Klick auf den Text oder das Icon springt die Anzeige auf die vorhergehende Seite der Historie.
Schreibweise: [BACK] eine Seite zurück [/BACK]
Anzeige: eine Seite zurück
Bemerkung: Anstelle des Textes kann auch eine Grafik eingefügt werden,
in der Form:

[BACK]{IMAGE:back_link_.gif}[/BACK]

[BOOKMARK] Text oder Bild [/BOOKMARK]

Verwendung: Erstellt einen Link zum Öffnen des Fensters
Zu Favoriten hinzufügen
Schreibweise: [BOOKMARK]Zu Favoriten hinzufügen[/BOOKMARK]
Anzeige:  
Alternative: Anstelle des Textes kann auch eine Grafik eingefügt werden in der Form:

[BOOKMARK]{IMAGE:bookmark_link_.gif}[/BOOKMARK]
 
 

[PRINT_PDF]Text oder Bild[/PRINT_PDF]

Verwendung:   Erstellt von der Aktuellen Seite eine PDF Datei. Diese Fumktion ist verfügar seit der Version 1.3.2
Schreibweise: [PRINT_PDF]PDF erstellen
Anzeige:
Alternative: Anstelle des Textes kann auch eine Grafik eingefügt werden,
in der Form:

[PRINT_PDF]{IMAGE:icon_pdf.gif}[/PRINT_PDF]
Bmerkung:  Die Ausgabe der PDF Datei können Sie über eine vorlagen/template Datei steuern.
Die Datei finden Sie im Ordner template/inc_default/pdf.tmpl

 

[PRINT] Text oder Bild [/PRINT]

Verwendung: Erzeugt einen Link zu einer druckerfreundlichen Ansicht, in der nur die Inhalts-Elemente des Content-Bereichs aufgeführt sind.

Alle Header-, Footer- und Navigationselemente sind entfernt.
Schreibweise: [PRINT]Druckansicht[/PRINT]
Anzeige:
Alternative: Anstelle des Textes kann auch eine Grafik eingefügt werden,
in der Form:

[PRINT]{IMAGE:print_link_.gif}[/PRINT]
 

[RSS all] Text oder Bild [/RSS]

Verwendung:   Mit diesem Replacement Tag können Sie Ihren Inhalt als RSS Feed anbieten
Schreibweise: [RSS all]RSS-Feed[/RSS]
Anzeige: RSS
Bemerkung:

Folgende Einstellungen sind möglich

[RSS all]RSS-Feed[/RSS]
gibt die komplette Webseite als RSS Feed aus.

[RSS INT]RSS-Feed[/RSS]
ersetzen Sie INT durch eine ID einer Strukturebene und es wird nur diese als RSS Feed ausgegeben.

[RSS ALIAS]RSS-Feed[/RSS]
ersetzen Sie ALIAS durch einen auf Ihrer Webseite bestehenden Alias und es wird nur diese Ebene als RSS Feed ausgegeben.
Alternative:

Anstelle des Textes kann auch eine Grafik eingefügt werden.

[RSS all]{IMAGE:rss_valid.gif}[/RSS]
 

{SITE}

Verwendung: Gibt die Aktuelle URL aus.
Schreibweise: {SITE}index.php?hallo
Anzeige: ./
Bemerkung: Kann vor allem bei Weiterleitungen benutzt werden. Sehr nützlich wenn man z.B. die Domain wechselt oder die Seite lokal entwirft.
