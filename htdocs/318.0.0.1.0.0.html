Ebene / Struktur kopieren

Um eine Ebene zu kopieren gehen Sie wie folgt vor.

Durch Klicken auf wählen Sie eine Ebene/Struktur zum Kopieren aus.

Um den Vorgang abzubrechen klicken Sie erneut auf das Icon in der ausgewählten Ebene.


Dort wo es nun möglich ist die Ebene einzufügen erscheint dieses Icon . 

Durch Klicken auf das Icon fügen Sie die Ebene (inkl. Artikeln) an ihrem neuen Ort ein.

Hinweis:

Die Aliase werden beim Kopieren nicht übernommen.


Um den Vorgang abzuschließen drücken Sie auf das Icon Ihrer verschobenen Ebene / Struktur.
