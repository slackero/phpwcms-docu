Content Part "Plain Text"

plain text (optional): Insert the text you wish to add here. You can insert several paragraphs one after another here. However, if you should need headings for other than the first paragraph, you should insert multiple “plain text” content-parts one after another into the article and in each case, use the fields "content title" and/or "sub-title". 

Special formatting in the text can be done by using the following codes:
[b]Text[/b] to make the enclosed text appear in bold-face.
[i]Text[/i] to make the enclosed text italicized.
[u]Text[/u] to make the enclosed text underlined.
