CVS Information

HINWEIS

Ab sofort wird auf der Docu Seite die aktuelle CVS Version von phpwcms bereitgestellt. Bitte beachten Sie dabei, dass dies dann grundsätzlich ein  Update ist und Sie somit darauf achten müssen, die Dateien an Ihr System anzupassen!

Dies trifft besonders für Dateien in den Ordnern

config/phpwcms/conf.inc.php
config/phpwcms/conf.template_default.inc.php
template/inc_css/frontend.css

sowie alle anderen Dateien, die Sie angepasst haben.

Die CVS Version finden Sie hier in 2 Varianten. Zum einen eine komplette Version und eine Version, die sich nur auf die Änderungen zur vorherigen Version bezieht. Sollten Sie nicht wissen, welche phpwcms Version sie zum Schluß installiert hatten, updaten Sie bitte Ihr phpwcms  erst einmal komplett. Danach können Sie dann die kleinen Updates benutzen und so Ihr phpwcms immer auf dem neuesten Stand halten.

Alle Änderungen, die im CVS vorgenommen wurden, finden Sie im folgenden Beitrag im Forum.

http://www.phpwcms.de/forum/viewtopic.php?t=8390
