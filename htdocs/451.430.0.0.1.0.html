System Add Ons

pixelTracking.php

Google Aanalytics und co einfach einbinden.

Mit dem Script pixelTracking.php haben Sie eine Einfache möglichkeit die Besucherzähler von Google Analytics (DE) , etracker (DE), phpMyvisits (FR/EN) und StatCounter (EN) einzubinden.

Bei allen den oben genannten Diensten müssen Sie sich voher Anmelden.

Öffnen Sie in einem Texteditor die Datei und editieren Sie die Einstellungen für den Service den Sie nutzen wollen.

Beispiel Google:

/*
* Google Aanalytics
*/
$_Tracking_GoogleAnalytics = Off; On = Google Analytics wird genutzt
$_Tracking_GoogleAnalyticsCode = 'UA-00000-1'; Tragen Sie hier Ihre ID ein
$_Tracking_GoogleSSL = Off; On = Wenn Sie die SSL Verbindung nutzen wollen.

makeEmailSpamSave.php

Ausgeschriebene Emailadressen sichern

Dieses Add On ist das dazu da, ausgeschriebene Email Adressen so verfälscht das Sie von Spam Bots nicht ausgelesen (genutzt) werden können.

fe_user.php

Fronend login Status

Diese Add On liefert Ihnen auf der Linken Seite Ihrer Webseite, sofern Sie sich vorher angemdet haben, ein Rotes Feld mit Ihrem Benutzernamen. Nun können Sie durch die Besucheransicht der Seite surfen. Und Können duch das klicken auf das Rote Feld sofort zu dem Artikel springen um ihn Bearbeiten.

lang_redirect.php

Sprachumschaltung

Mit dieser erweiterung die Seit der Version vom 5.12.2007 vorhanden ist. Ist es möglich eine Automatische Sprachweiterleitung zu definieren.

Passen Sie die Datei die unter template/inc_script/frontend_init/disabled/ liegt, an Ihre Seite an. Und verschieben sie die Datei nach template/inc_script/frontend_init/

Als Beispiel:

Der Englische Inhalt hat die ID 1
Der Deutsche Inhalt hat die ID 2

Müssen die Zeilen wie folgt aussehen.

'www.ihre_domain_tld' => array( 'ID' => 1, 'LANG' => 'EN', 'HOME_URL' => 'http://www.ihre_domain.tld/Ihr_alias_zum_englischen_inhalt' ),

'www.ihre_domain.tld' => array( 'ID' => 2, 'LANG' => 'DE', 'HOME_URL' => 'http://www.ihre_domain.tld/index.php?Ihr_alias_zum_deutschen_inhalt' ),

Es können beliebig viele Sprachumleitungen definiert werden. Der Erste Eintrag wird immer als Standard definiert falls keine der Sprache auf den Besucher zutrifft.
