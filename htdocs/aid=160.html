Datenbank wiederherstellen

Um die Datenbank wiederherzustellen können Sie das von den meisten Providern zur Verfügung gestellte phpMyAdmin benutzen. Wie der Pfad zu diesem Programm ist erfahren Sie von Ihrem Provider.

Die folgenden Screenshots können sich je nach Version von phpMyAdmin unterscheiden. Die Vorgehensweise bleibt jedoch identisch.

Durch Drücken auf SQL gelangen Sie zur Eingabemaske, in der SQL Befehle direkt in der Datenbank ausgeführt werden können. Dazu gehört auch das Wiederherstellen der Datenbank.

Wie Sie sehen ist die Maske in 2 Teile unterteilt. Im oberen Bereich können Sie SQL-Befehle direkt in die Datenbank schreiben. 

Wiederherstellen durch Kopieren und Einfügen:

Damit ist auch ein Wiederherstellen der Datenbank möglich. Das ist jedoch nur notwendig wenn Ihre Datenbank die Größe von 2 MB überschreitet(Die mögliche Größe der SQL-Datei wird von Ihrem Provider vorgegeben. Sie finden diese meist hinter dem Feld Durchsuchen wenn Sie die Datenbank mit der SQL-Datei wiederherstellen wollen) und ein Wiederherstellen über die SQL-Datei nicht mehr möglich ist. Dafür müssen Sie jedoch die ZIP-Datei, die die SQL-Datei enthält, entpacken. Dann öffnen Sie die SQL-Datei mit Wordpad oder einen anderen Texteditor und markieren mehrere Zeilen und fügen diese via Kopieren und Einfügen in das Textfeld in Ihrem phpMyAdmin ein. Durch Drücken auf OK werden die Zeilen in die SQL-Datenbank geschrieben.

Dieser Schritt ist auch notwendig bei einem Upgrade von Version 1.1-RC4 auf die (Dev.) 1.2.x

Wiederherstellen durch die SQL Datei:

Hinweis: Bitte beachten Sie die maximale Größe der SQL-Datei hinter dem Feld Durchsuchen. Wenn Ihre Datei diese Größe überschreitet, teilen Sie Ihre SQL-Datei in ca. 2 MB große Teile  auf oder nutzen Sie die zuerst beschriebene Möglichkeit.

Klicken Sie auf Durchsuchen und wählen Sie die SQL-Sicherungsdatei aus. Wählen Sie darunter die Komprimierung aus, meist ist es GZip. 
Dann drücken Sie auf OK. Nun sollte Ihre Datenbank wiederhergestellt werden. Sollte es zu einer Fehlermeldung kommen probieren Sie den gleichen Schritt nochmal. Aber entpacken Sie diesmal die ZIP-Datei auf Ihrem Computer, Sie erhalten dann eine Datei mit der Endung .sql . Wählen Sie diese dann beim erneuten Klick auf Durchsuchen aus und wählen Sie Keine Kompression.
