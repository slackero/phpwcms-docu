Content Part "HTML"

click to enlarge
plain html (optional): You can place any HTML code in this field, which will be displayed in the appropriate place in the article. This makes phpWCMS very flexible. One can insert a form in an individual article for instance, which passes the form data to an external script. For everyday management, however, this type of content-part is not
usually needed. A good introduction to HTML can be found at
http://www.w3schools.com/html/
