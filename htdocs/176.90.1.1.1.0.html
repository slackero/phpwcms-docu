Replacement Tags

Nun haben wir Schriftarten, Farben und Stile definiert, doch wie kann man dies jetzt in phpwcms einsetzen ?

Dafür hat das GT MOD einen Replacement Tag integriert, mit dem man jetzt relativ einfach die Stile einsetzen kann.

{GT:stil}Mein Text{/GT}

Verwendung: Wendet einen im Backend definierten Stil auf einen beliebigen Text an
Schreibweise: {GT:Stil} Mein Text {/GT}
Anzeige:
Bemerkung: Der definierte Stil wird auf den Text angewandt. Der Tag kann auch als Text einen weiteren Tag enthalten, z.B. Author :
{GT:Stil2} {AUTHOR} {/GT}
Anzeige:
 

Zur Beachtung für alle copy & paste Fans !

Um die Schreibweise der Replacement-Tags innerhalb diese Beitrags verdeutlichen zu können, ohne daß sie vom Programm sofort intepretiert wurden, mußte innerhalb der einzelnen Titel und im Feld Schreibweise hinter der ersten öffnenden Klammer ein SPACER gesetzt werden, der im RT natürlich nicht enthalten sein darf! 
( <img src="img/leer.gif" border="0" width="1" height="1" alt="" /> ) 

Werden also RTs aus dieser Seite kopiert, ist die Funktion nicht sichergestellt!
Es wird dringend geraten, die Replacement-Tags von Hand in den eigenen Code einzugeben oder den SPACER-Code zu entfernen!
