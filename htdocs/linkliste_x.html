Content Part "Linkliste"

Linkliste (optional): Geben Sie hier nach dem folgenden Muster die gewünschten Links ein:

Bezeichnung|Adresse

Geben Sie für jeden Link eine neue Zeile ein. Wenn Sie auf eine Mailadresse verlinken wollen, geben Sie vorher „mailto:“ ein, also z.B. „mailto:Info@MeineDomain.de“.

Eine Linkliste könnte beispielsweise so aussehen:

Info@MeineDomain.de|mailto:Info@MeineDomain.de
www.MeineDomain.de|http://www.MeineDomain.de

Die Ausgabe wäre in diesem Falle folgende:

Info@MeineDomain.de
www.MeineDomain.de 

Um einen Link in einem neuen Browserfenster zu öffnen sieht der Linkaufbau wie folgt aus.

www.MeineDomain.de|http://www.MeineDomain.de _blank
