Die conf.inc.php

Die conf.inc.php enthält alle Systemeinstellungen. Die Werte können Sie nach Ihren Wünschen anpassen.
Zeile 26-31 -> Datenbank Einstellungen
26 $phpwcms["db_host"] = "localhost"; Tragen Sie hier den Host Ihrer Datenbank ein.
27 $phpwcms["db_user"] = ""; Tragen Sie hier den Benutzernamen der Datenbank ein
28 $phpwcms["db_pass"] = ""; Tragen Sie hier das Passwort zu Ihrer Datenbank ein.
29 $phpwcms["db_table"] = ""; Tragen Sie hier den Tabellennamen ein.
30 $phpwcms["db_prepend"] = ""; Falls bei der Installation ein Präfix für die Datenbank-Tabellen angegeben wurde, muss dieses nun hier stehen.
31 $phpwcms["db_pers"] = 1; Soll die Datenbankverbindung durchgehend bestehen bleiben (Default: 1) oder vor jedem Anfrage neu erstellt und anschliessend wieder getrennt werden (0). Empfohlen wird diesen Wert auf 1 gesetzt zu lassen.
 
Zeile 34-45 -> Webseiten Einstellungen
34 $phpwcms["site"] = "http://www.phpwcms.de/"; Tragen Sie hier Ihren Domainnamen dieser muss immer mit einem / enden.
35 $phpwcms["admin_email"] = ""; Tragen Sie hier die E-Mail Adresse des Administrators / Webmasters ein
 
Zeile 38-46 -> Pfade
38 $phpwcms["root"] = ""; Wenn Sie phpwcms in ein Unterverzeichnis installieren tragen Sie hier den Namen des Unterverzeichnisses ein.
39 $phpwcms["file_path"] = "phpwcms_filestorage"; Pfad zum Filestorage Verzeichnis. In diesem werden die Ordner und Dateien der einzelnen Benutzer von phpwcms angelegt
40 $phpwcms["file_tmp"] = "phpwcms_tmp"; Name des tmp Ordner
41 $phpwcms["templates"] = "phpwcms_template"; Name zum Template Ordners
42 $phpwcms["dir_thlist"] = "thumb_list"; Name de Thumbnail Ordners
43 $phpwcms["dir_preview"] = "thumb_preview"; Name des Preview Ordners
44 $phpwcms["content_path"] = "content"; Name des Content Ordners
45 $phpwcms["cimage_path"] = "images"; Name des (Content) Image Ordners
46 $phpwcms["ftp_path"] = "phpwcms_ftp"; Name des ftp Ordners
 
Zeile 49-55 -> Content Einstellungen
49 $phpwcms["file_maxsize"] = 52428800; //Bytes (50 x 1024 x 1024) Hier bestimmen Sie die maximale Dateigröße
50 $phpwcms["content_width"] = 538; Hier bestimmen Sie die Breite des Contentbereiches in Pixeln
51 $phpwcms["img_list_width"] = 100; Hier bestimmen Sie die maximale Höhe des Vorschaubildes in Pixeln 
52 $phpwcms["img_list_height"] = 75; Hier bestimmen Sie die maximale Breite des Vorschaubildes in Pixeln
53 $phpwcms["img_prev_width"] = 538; Hier bestimmen Sie die maximale Breite  in Pixeln
54 $phpwcms["img_prev_height"] = 538; Hier bestimmen Sie die maximale Höhe des Vorschaubildes in Pixeln 
55 $phpwcms["max_time"] = 1800; Hier bestimmen Sie die Zeit nach der Sie automatisch aus dem Administrationsbereich ausgeloggt werden. Standard ist 1800 = 30 min
 
Zeile 58-72 -> verschiedenes
58 $phpwcms["compress_page"] = 3; Hier können Sie die Seitenkompression einschalten.
Wert: 0 = aus / 1-9 = an / 9 = höchste Kompression
59 $phpwcms["imagick"] = 0; //if 0 = GD, 1 = ImageMagick convert, 2 = ImageMagick 4.2.9 Hier bestimmen Sie welche Grafikbibliothek Sie benutzen wollen (GD ist Bestandteil von PHP). Einige Provider bieten auch ImageMagick an. Dann haben Sie folgende Möglichkeiten: Wert 0 = GD1 / 1 = ImageMagick convert / 2 = ImageMagick 4.2.9
60 $phpwcms["imagick_path"] = ""; //Path to ImageMagick Wenn Sie ImageMagick verwenden können Sie hier den Pfad eintragen. Meist ist es /usr/bin oder /usr/bin/convert.
61 $phpwcms["use_gd2"] = 1; //if 0 = GD1, 1 = GD2 Hier bestimmen Sie welche GD Bibliothek verwendet werden soll.
Wert: 0 = GD1 / 1 = GD2
62 $phpwcms["rewrite_url"] = 0; Hier können Sie das REWRITE einschalten. Dazu muss jedoch im Hauptverzeichnis Ihrer phpwcms Installation die _.htaccess in .htaccess umbenannt werden.
Wert: 0 = aus / 1 = an
63 $phpwcms["wysiwyg_editor"] = 1; Hier können Sie bestimmen welcher WYSIWIG Editor bei der Artikelerstellung benutzt werden soll.
Wert: 0 = kein Editor / 1 = HTMLarea / 2 = FCKeditor / 3 = Browser basierend / 4 = Spaw
64 $phpwcms["phpmyadmin"] = 1; Hier können Sie bestimmen ob im Administrator Bereich der Link zu phpMyAdmin angezeigt wird. Wert: 0 = aus / 1 = an 
65 $phpwcms["default_lang"] = "en"; Hier bestimmen Sie die Standardsprache für den Verwaltungsbereich (Backend). Es können nur die Sprachen verwendet werden in denen phpwcms verfügbar ist. Eine aktuelle Liste finden Sie hier. Spracheinstellungen
66 $phpwcms["charset"] = "iso-8859-1"; Hier bestimmen Sie den Zeichnsatz den Sie verwenden wollen. Die Standardeinstellung ist hier iso-8859-1. Eine Übersicht über alle ISO Zeichensätze finden Sie hier (engl.) Zeichensätze
67 $phpwcms["allow_remote_URL"] = 0; Wenn Sie im {PHP:...} Tag URL aufrufen wollen müssen Sie diesen Wert auf 1 setzten. Wert: 0 = URL erlaubt (standard) / 1 = URL verboten
68 $phpwcms["gt_mod"] = 0; Hier können Sie den Graphical Text aktivieren. Wert: 0 = aus (Standard) / 1 = an
69 $phpwcms["jpg_quality"] = 75; Hier können Sie die jpg Qualität einstellen in %.
Wert: 25 -100 / Standard = 75
70 $phpwcms["sharpen_level"] = 1; Wenn Sie ImageMagick nutzen haben Sie hier die Möglichkeit Ihre Bilder noch extra zu schärfen. Wert: 0 = aus / 1 - 5 = leicht - extra schärfen
71 $phpwcms["allow_ext_init"] = 0; Soll es erlaubt sein im /template/inc_script/frontend_init Ordner von phpwcms alle Scripts vor dem Laden des Frontends zu laden (nützlich für eigene PHP-Funktionen). Wert:  0 - verboten (Standard) / 1 - erlaubt
72 $phpwcms["allow_ext_render"] = 0; Soll es erlaubt sein die im Init Folder geladenen Funktionen nun auch beim Rendern der Seite einzusetzen (/template/inc_script/frontend_render). Wert:  0 - verboten (Standard) / 1 - erlaubt
 
Zeile 75-77 -> SSL Einstellungen
75 $phpwcms["site_ssl_mode"] = "0"; Im Moment noch nicht möglich
76 $phpwcms["site_ssl_url"] = ""; Im Moment noch nicht möglich
77 $phpwcms["site_ssl_port"] = "443"; Im Moment noch nicht möglich
 
Zeile 80-87 // SMTP Einstellungen
80 $phpwcms['SMTP_FROM_EMAIL'] = 'info@localhost'; Tragen Sie hier die E-mail Adresse ein die angezeigt werden soll wenn eine Mail über ein Kontaktformular verschickt wird.
81 $phpwcms['SMTP_FROM_NAME'] = 'My Name'; Tragen Sie hier den Namen ein der angezeigt werden soll wenn eine Mail über ein Kontaktformular verschickt wird.
82 $phpwcms['SMTP_HOST'] = 'localhost'; Tragen Sie hier den SMTP Host ein um Mails via SMTP zu versenden
83 $phpwcms['SMTP_PORT'] = 25; Tragen Sie hier den Port für das Versenden von Mails via SMTP ein. Der Standard ist meist 25.
84 $phpwcms['SMTP_MAILER'] = 'mail'; Hier bestimmen Sie womit die Mails gesendet werden. Wert:  smtp = via SMTP / mail = phpmailer (standard) / sendmail = via sendmail
85 $phpwcms['SMTP_AUTH'] = 0; Wenn Sie sich an Ihrem Mailserver anmelden müssen Sie den Wert auf 1 setzen ansonsten bleibt der wer auf 0
86 $phpwcms['SMTP_USER'] = 'user'; Tragen Sie hier den Benutzernamen für die SMTP Benutzung ein.
87 $phpwcms['SMTP_PASS'] = 'pass'; Tragen Sie hier das dazugehörige SMTP Passwort ein.
