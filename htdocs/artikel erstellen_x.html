Artikel erstellen

Um neue Artikel zu erstellen klicken Sie in der Artikelzentrale entweder auf das Symbol oder in der linken Navigation auf "Neuer Artikel".

In dem erscheinenden Bereich können Sie die Basisinformationen für den neuen Artikel vornehmen:

 
  Feldbeschreibungen
Kategorie: Wenn Sie auf "Neuer Artikel" geklickt haben, können Sie hier aus dem Pull-Down-Menü die Strukturebene wählen, unter der der Artikel erscheinen soll. Wenn Sie auf das Symbol geklickt haben, um diesen Artikel zu erstellen, ist die entsprechende Strukturebene bereits gewählt. Sie können Sie aber auch dann noch über das Pull-Down-Menü anpassen.
Alias ID(ab ver. 1.2.5): Durch Eintragen der Artikel ID eines anderen Artikels ist es möglich, einen Artikel an anderer Stelle erneut anzeigen zu lassen. Die Artikel ID finden Sie immer in [ID:XXX] hinter der Kategoriebeschreibung innerhalb der Artikelinformation oder auch wenn sie mit dem Mauszeiger über folgendes Symbol fahren
Artikeltitel: Die Hauptüberschrift des Artikels. Mit der Option "versteckt" können Sie die Anzeige der Überschrift für den Besucher der Website unterdrücken.
Versteckt: Setzen Sie hier den Haken, wird die Artikelüberschrift nicht angezeigt.
Untertitel (optional): Eine Unterüberschrift des Artikels.
Anzeige von ... bis (optional): Die Veröffentlichung des Artikels auf einen bestimmten Zeitraum beschränken. Geben Sie den Zeitpunkt jeweils in der angegebenen Form an, also z.B. "2004-05-01 12:00:00" für den Mittag des 1. Mai 2004.
Weiterleiten (optional): Beim Aufruf des Artikels den Betrachter automatisch auf eine andere Seite weiterleiten. Sie können hier z.B. eine externe Adresse (z.B. http://www.phpwcms-docu.de/) angeben oder mittels der Direktlinks innerhalb der Website weiterleiten (z.B. "index.php?direktlink"). Wir empfehlen bei weiterleiten den tag {SITE} zu nutzen.
Schlüsselwört. (optional): Eingetragene Schlüsselwörter werden von der Suchfunktion innerhalb der Website gefunden.
Cache: Her können Sie die Cache funktion für den Artikle verändern.
Suche: Standard ist, dass der gesamte Inhalt durch die Suche gefunden wird. Entfernen Sie den Haken, wird dieser Artikel von der Suche ausgeschlossen.
Sichtbar / Öffentlich: Damit Besucher der Website den Artikel sehen können, müssen „sichtbar“ und „öffentlich“ gesetzt sein. Die hier gesetzte Markierung erscheint in der Artikelzentrale als und – nicht gesetzte Markierungen als und (siehe auch im Abschnitt „Die Artikelzentrale“ weiter oben). TIPP: Wenn Sie einen neuen Artikel erstellen, bietet es sich an, diesen zunächst unsichtbar zu halten bis alle Inhalte eingepflegt sind, da sonst Besucher der Website den unvollständigen Artikel bereits sehen können.
Sitemap: Normal wird ein Artikel in der Sitemap angezeigt. Durch Entfernen des Hakens können Sie das verhindern.
[WEITER...] Link:(ab ver. 1.2.5): Wenn Sie mehrere Artikel in einer Struktur haben, können Sie hier bestimmen, ob der Weiter... Link nach dem Schlagtext angezeigt werden soll.



Nachdem Sie Ihre Einstellungen vorgenommen haben, bestätigen Sie diese mit einem Klick auf den Button „Neuen Artikel erstellen“.

Artikel Erstellen

Im oberen Bereich sehen Sie erneut Ihre zuvor getätigten Eingaben in einer Übersicht. Zusätzlich können Sie nun weitere Einstellungsmöglichkeiten vornehmen:
  Feldbeschreibungen
Vorlage:
Artikellisting Hier können Sie eine Vorlage für das Artikellisting auswählen. Wählen Sie zum Test das sample.tmpl in dem Auswahlmenue aus. Die Vorlage für das Artikellisting finden Sie in folgendem Ordner /template/inc_cntpart/articlesummary/list/sample.tmpl
max. Anzahl Worte
(Summary)		 Geben Sie hier die Anzahl der Worte ein die vom Schlagtext angezeigt werden sollen. Wenn Sie diese Option verwenden. Sollten Sie die Option "Schagtext nicht Anzeigen" nicht Aktivieren. Da es sonst möglich wäre das nicht der Komplette Schlagtext gelesen werden kann.
[WEITER...] LINK Wenn Sie die Option wählen wird am Ende des Schlagtextes ein Link angezeigt. Standardmäßig wird more ... angezeigt. In der Datei (confing/phpwcms/)conf.template_default.php unter $template_default['top_readmore_link']='more&#8230;'; können Sie das Aussehen bzw. den more text abändern
Artikeldetail Hier können Sie eine Vorlage für den Artikel auswählen. wählen Sie zum Test das sample.tmpl in dem Auswahlmenue aus. Die Vorlage für der Artikel finden Sie in folgendem Ordner /template/inc_cntpart/articlesummary/article/sample.tmpl.
Schlagtext nicht anzeigen Wenn Sie diese Option verwenden Wird der Schlagtext nicht im kompletten Artikel angezeigt.
Content Parts paginieren

Hiermit ist es möglich Content Part zu paginieren. D.h. jeder Content Part ist in diesem Artikel eine Einzelne Seite innerhalb des Artikels.
Dazu erscheint dann im oberen rechtem Bildrand eine zweite Navigation.
Schlagtext Eine Zusammenfassung des Artikels. Diese erscheint in Artikellisten (z.B. der Suche) und als Einleitung des Artikels auf der Artikelseite.
Artikeldetail Hier können Sie ein Bild auswählen welches im Artikeldetail erscheint.
Artikellisting Hier können Sie ein Bild auswählen welches im Artikellisting erscheint. Wenn Sie keines Auswählen wird das Bild vom Artikeldetail übernommen.
Artikelbesitzer Hier wird der Besitzer des Artikels bestimmt. Sofern Sie kein Admin sind kann nur derjenige den Artikel bearbeiten der dort steht. Nur der Administrator kann den Artikelbesitzer verändern.
Author Hier kann man den Namen des Autors des Artikels angeben. Wird dort nichts eingetragen wird automatisch der Artikelbesitzer eingetragen.
Cache Hier können Sie die Cache funktion für den Artikle verändern.
Suche Standard ist, dass der gesamte Inhalt durch die Suche gefunden wird. Entfernen Sie den Haken, wird dieser Artikel von der Suche ausgeschlossen.
Sichtbar / Öffentlich: Damit Besucher der Website den Artikel sehen können, müssen „sichtbar“ und „öffentlich“ gesetzt sein. Die hier gesetzte Markierung erscheint in der Artikelzentrale als und – nicht gesetzte Markierungen als und (siehe auch im Abschnitt „Die Artikelzentrale“ weiter oben). TIPP: Wenn Sie einen neuen Artikel erstellen, bietet es sich an, diesen zunächst unsichtbar zu halten bis alle Inhalte eingepflegt sind, da sonst Besucher der Website den unvollständigen Artikel bereits sehen können.
Sitemap: Normal wird ein Artikel in der Sitemap angezeigt. Durch Entfernen des Hakens können Sie das verhindern.
[WEITER...] Link:(ab ver. 1.2.5): Wenn Sie mehrere Artikel in einer Struktur haben, können Sie hier bestimmen, ob der Weiter... Link nach dem Schlagtext angezeigt werden soll.

Artikel setzen sich, wie oben bereits erläutert, aus Inhalten zusammen. Dies können Texte, Bilder, Links und andere Bestandteile einer Webseite sein. In phpWCMS können Sie Inhalte wie Bausteine zusammenfügen, um den gewünschten Artikel zu erstellen.

Mithilfe des Pull-Down-Menüs im unteren Bereich können Sie Inhalte hinzufügen. Wählen Sie einfach den zu erstellenden Inhaltstyp (Content Part) aus und klicken Sie anschließend auf "add content part" sofern Ihr Browser Sie nicht automatisch weiterleitet.
