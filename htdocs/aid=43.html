Replacement Tags für Links

RT: {Keyword:Schlüsselwort}
Verwendung: Erzeugt einen Link / Linkliste für ein bestimmtes Artikel Schlüsswort. Im Beispiel "Tags"
Schreibweise: {KEYWORD:Tags}
Ausgabe-Format: Tags
Bemerkung:

Kommt das Schlüsselwort in mehreren Artikeln vor wird eine Liste erstellt.

Siehe Beispiel:
{KEYWORD:replacement tags}
replacement tags [1|2]
RT: [LINK_IT] {WONG} {DIGG} {DELICIOUS} {TECHNORATI} {SPURL} {FURL} {YAHOO}[/LINK_IT]
Verwendung: Erzeugt Link's zu den Bookmarkseiten Mister Wong, Digg, del.icio.us, Technorati, Spurl, Furl und Yahoo
Schreibweise: [LINK_IT] {WONG} {DIGG} {DELICIOUS} {TECHNORATI} {SPURL} {FURL} {YAHOO} {GOOGLE} {MAGNOLIA} {NEWSVINE} {REDDIT}[/LINK_IT]
Ausgabe-Format:
Bemerkung:

Dieser Replacement Tag ist ab der Version 1.2.9 verfügbar und es muss dazu das linkme.php script im template/inc_script/frontend_render liegen.

Durch entfernen von einzelnen Bookmarktags ({WONG} {DIGG} {DELICIOUS} {TECHNORATI} {SPURL} {FURL} {YAHOO} {GOOGLE} {MAGNOLIA}  {NEWSVINE} {REDDIT}) können Sie entscheiden zu welchem Service Sie einen Link bereitsellen wollen.
RT: [MAIL eMail_Adresse] Text [/MAIL]
Verwendung: Öffnet das aktuelle eMail Programm des Users und setzt die angegebene eMail-Adresse ein
Schreibweise: [MAIL eMail_Adresse] Text [/MAIL]
Ausgabe-Format: eMail senden
Bemerkung:

Anstatt des Textes kann hier auch der {IMAGE:xxx.xx} Tag zum benutzen einer Grafik verwendt werden.
RT: [MAILSUB eMail_Adresse Betreff] Text [/MAILSUB]
Verwendung: Öffnet das aktuelle eMail-Programm des Users und setzt die angegebene eMail-Adresse und den Betreff-Text ein
Schreibweise: [MAILSUB docu@phpwcms.org BETREFF] eMail und Betreff [/MAILSUB]
Ausgabe-Format: eMail und Betreff
Bemerkung:

Ersetzen Sie einfach das Wort BETREFF mit dem Wort, das im Betreff erscheinen soll.

Anstatt des Textes kann hier auch der {IMAGE:xxx.xx} Tag zum Benutzen einer Grafik verwendet werden.
RT: [TOP] Text [/Top]
Verwendung: Erzeugt einen Link zum Anfang der aktuellen Seite
Schreibweise: [TOP] Text [/Top]
Ausgabe-Format: zum Seitenanfang
Bemerkung:

Anstelle des Textes kann auch eine Grafik eingefügt werden,
in der Form:

[TOP]{IMAGE:top_link_.gif}[/Top]
RT: {NEW:Anzahl_angezeigter_Links}
Verwendung: Anzeige einer Liste (mit Datum und Artikel-Titel) der zuletzt bearbeiteten Artikel, deren Anzahl angegeben werden kann.
Schreibweise: {NEW:5}
Ausgabe-Format:

17/11/2010 - Templates

02/06/2009 - HowTo

02/06/2009 - HowTo

01/01/2009 - DEV - Version ZIP Format

01/01/2009 - DEV - Versionen im ZIP Format
Bemerkung:

Der zuletzt bearbeitete Artikel steht in der Liste oben.
Um das Ausgabeformat de Datums  zu ändern schaun Sie bitte hier vorbei. Spracheinstellungen

Das Selektionskriterium wird in der Datei "conf.template_default.inc.php" angegeben unter
$template_default['news']['sort_by']      = 'cdate';
// 'cdate' = Creation date, or 'udate' = update date, ldate = start date, kdate = end date

  • cdate: Erstellungsdatum
  • udate: Update-Datum
  • ldate: Startdatum
  • kdate: Enddatum
RT: {NEW:Anzahl_angezeigter_Links:ID}
Verwendung: Anzeige einer Liste (mit Datum und Artikel-Titel) der zuletzt bearbeiteten Artikel, einer Strukturebene (ID), deren Anzahl angegeben werden kann.
Schreibweise: {NEW:5:1}
Ausgabe-Format:

16/08/2004 - {docu} phpwcms {/docu}
Bemerkung:

Der zuletzt bearbeitete Artikel steht in der Liste oben.
Um das Ausgabeformat de Datums  zu ändern schaun Sie bitte hier vorbei. Spracheinstellungen

Das Selektionskriterium wird in der Datei "conf.template_default.inc.php" angegeben unter
$template_default['news']['sort_by']      = 'cdate';
// 'cdate' = Creation date, or 'udate' = update date, ldate = start date, kdate = end date

  • cdate: Erstellungsdatum
  • udate: Update-Datum
  • ldate: Startdatum
  • kdate: Enddatum
RT: {RELATED:Anzahl_angezeigter_Links:Schluesselwort}
Verwendung: Erstellt eine Liste mit Artikeln-Titeln, basierend auf einem angegebenen Schlüsselwort. Die Maximalzahl aufgeführter Artikel kann angegeben werden.
Schreibweise: {RELATED:5:bearbeitet}
Ausgabe-Format:

Die Artikelzentrale

Replacement-Tags

Replacement Tags für die Navigation

Die Organisation von Artikeln in Strukturebenen

Neue Artikel erstellen
Bemerkung:

Es werden nur die Artikel aufgeführt, die innerhalb der Artikel- Basisinformation im Feld Schlüsselwörter den gesuchten Eintrag besitzen. Es wird nicht nach Groß- und Kleinschreibung unterschieden!
RT: [INT url] Text [/INT]
Verwendung: Erzeugt einen internen Link im aktuellen Fenster
Schreibweise: [INT index.php?die_docu]Die Docu[/INT]
Ausgabe-Format: Die Docu
Bemerkung:

Die verlinkte Seite (Kategorie) Die Docu hat in diesem Beispiel den Alias-Namen: die_docu
RT: [EXT url] Text [/EXT]
Verwendung: Erzeugt einen externen Link in einem neuen Fenster
Schreibweise: [EXT http://www.phpwcms.de]phpwcms[/EXT]
Ausgabe-Format: phpwcms
Bemerkung:

--
RT: [ID integer] Text [/ID]
Verwendung: Erzeugt einen internen Link zu einem Artikel, der durch seine Artikel-ID angegeben wird.
Schreibweise: [ID 7]Sitemap[/ID]
Ausgabe-Format: Sitemap
Bemerkung:

Der verlinkte Artikel Sitemap hat im Beispiel die Artikel-ID: 7
RT: [ID alias] Text [/ID]
Verwendung: Erzeugt einen internen Link zu einer Seite (Kategorie), die durch ihren Alias-Namen angegeben wird.
Schreibweise: [ID suche]Suche[/ID]
Ausgabe-Format: Suche
Bemerkung:

Die verlinkte Seite (Kategorie) Suche hat in diesem Beispiel den Alias-Namen: suche
