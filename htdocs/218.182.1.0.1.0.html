Modules

The functionality of phpwcms further can be expanded through characteristic Module. In the current version (1.1 RC 4), the Graphical text MOD of Jérôme Gamez in the system is integrated. To use around this module it explicitly must be disconnected in the (include/inc_conf/) conf. inc. php


$phpwcms["gt_mod"] = 1;  //0 = Graphical Text MOD disabled, 1 = enabled 


Because phpwcms is yet in a stage, in which at the internal construction of the system yet variations are made, it is appear advisably with the development from MODs to that to wait the next version. Consider also that possible changes make an upgrade at the system on a newer version hard.

Here it goes to the official MOD list in the phpwcms forum. Please you note that in the frame of this documentation only Module are described, that are firm component of phpwcms!

In the following the Graphical text MOD will be explicitly discribed. If you wanna create a modul by your own, please read this.
